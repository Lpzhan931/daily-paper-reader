## 摘要
投机解码（Speculative decoding）加速了大语言模型（LLM）的推理，但当目标模型针对特定领域进行微调时，其性能会发生退化。一种朴素的解决方案是为每个目标模型重新训练草稿模型，但这既昂贵又低效。为了解决这一问题，我们引入了一个名为“高效草稿适配”（Efficient Draft Adaptation，简称 EDA）的参数和数据高效框架，用于高效地适配草稿模型。EDA 引入了三项创新：(1) 一种解耦架构，利用共享和私有组件分别对共享和目标特定的输出分布进行建模，通过仅更新轻量级的私有组件来实现参数高效的适配；(2) 一种数据再生策略，利用微调后的目标模型重新生成训练数据，从而改善训练与投机解码之间的对齐，进而提高平均接受长度；(3) 一种样本选择机制，优先选择高价值数据以实现高效适配。我们的实验表明，EDA 有效地恢复了微调模型上的投机性能，与完全重新训练相比，在显著降低训练成本的同时，实现了更优的平均接受长度。代码可在 https://github.com/Lyn-Lucy/Efficient-Draft-Adaptation 获取。

## Abstract
Speculative decoding accelerates LLM inference but suffers from performance degradation when target models are fine-tuned for specific domains. A naive solution is to retrain draft models for every target model, which is costly and inefficient. To address this, we introduce a parameter- and data-efficient framework named Efficient Draft Adaptation, abbreviated as EDA, for efficiently adapting draft models. EDA introduces three innovations: (1) a decoupled architecture that utilizes shared and private components to model the shared and target-specific output distributions separately, enabling parameter-efficient adaptation by updating only the lightweight private component;(2) a data regeneration strategy that utilizes the fine-tuned target model to regenerate training data, thereby improving the alignment between training and speculative decoding, leading to higher average acceptance length;(3) a sample selection mechanism that prioritizes high-value data for efficient adaptation. Our experiments show that EDA effectively restores speculative performance on fine-tuned models, achieving superior average acceptance lengths with significantly reduced training costs compared to full retraining. Code is available at https://github.com/Lyn-Lucy/Efficient-Draft-Adaptation.

#### 1. 核心问题与整体含义
*   **研究动机**：投机解码（Speculative Decoding）通过轻量级“草稿模型”预先生成候选 Token 并由“目标模型”验证来加速推理。然而，当目标模型针对特定领域（如数学、代码、医疗）进行监督微调（SFT）后，其输出分布会发生显著偏移。
*   **核心问题**：预训练的草稿模型与微调后的目标模型不再匹配，导致 Token 接受率骤降，加速效果丧失。若为每个微调后的模型重新训练专用草稿模型，计算成本极其高昂。
*   **整体含义**：本文提出了 **EDA (Efficient Draft Adaptation)** 框架，旨在通过极小的参数更新和数据消耗，快速将现有草稿模型对齐到新的目标模型，恢复投机采样的加速性能。

#### 2. 方法论
EDA 框架由三个核心模块组成：
*   **共享-私有解耦架构 (Shared-Private Gated Architecture)**：
    *   将草稿模型的 FFN 层替换为两个专家：**共享专家 (Shared Expert)** 继承自通用预训练模型并冻结，负责建模通用的语言规律；**私有专家 (Private Expert)** 是轻量级的，负责捕捉特定领域的分布偏移。
    *   引入门控机制动态融合两者的输出，适配时仅更新私有专家和门控参数。
*   **领域特定自生成策略 (Domain-Specific Self-Generation)**：
    *   为了消除训练数据与目标模型实际输出之间的不一致（Mismatch），EDA 让微调后的目标模型在领域数据集上进行自回归生成，以此作为草稿模型的训练监督信号。
*   **基于表示偏移的样本选择 (Data-Efficient Selection)**：
    *   利用 PCA（主成分分析）降低目标模型隐藏层表示的维度。
    *   计算样本与通用分布之间的**马氏距离 (Mahalanobis Score)**，识别出那些偏离通用的、具有高适配价值的“难样本”。
    *   仅使用前 50% 的高价值样本进行训练，进一步降低计算开销。

*   **性能恢复**：EDA 在所有领域均显著提升了平均接受长度（$\tau$）。例如在数学任务上，$\tau$ 从 1.17（未适配）提升至 4.79，甚至超过了全参数微调（4.37）。

#### 7. 优点
*   **设计巧妙**：通过解耦架构解决了“灾难性遗忘”与“适配成本”的平衡问题。

*   **落地性强**：在有限算力（4张 GPU）和短时间内（2小时）即可完成适配，非常适合模型频繁迭代的工业场景。

*   **模型架构限制**：实验主要基于 Qwen2.5-7B 系列，对于更大规模（如 70B+）或不同架构（如 MoE 目标模型）的适配效果尚待验证。
