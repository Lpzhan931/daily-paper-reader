## 摘要
上下文稀疏性是减少大语言模型（LLM）推理过程中计算复杂度的方法之一。现有的基于上下文稀疏性且精度损失极小的 LLM 推理加速技术通常需要训练稀疏模式预测器。本文提出了一种用于加速 LLM 中基于 ReGLU 的前馈网络（FFN）推理的框架。该框架提供了一种快速、无需训练的方法，通过对门控投影矩阵进行截断感知的奇异值分解（SVD）来构建稀疏模式预测器，并配备了阈值校准算法以及支持在 CUDA 和 CANN 设备上进行条件计算的推理执行器。在三个 FFN 平均激活稀疏度为 90% 的稀疏 LLM 上进行的实验表明，端到端解码时间缩短了高达 1.8 倍，同时在涉及复杂数学和代码生成的任务基准测试中，评分下降不到 1%。这项工作推进了 LLM 在边缘设备上的部署。

## Abstract
Contextual sparsity is one of the approaches used to reduce computational complexity in the inference process of large language models (LLMs). Existing techniques for efficient LLM inference acceleration based on contextual sparsity with minimal accuracy degradation require training sparse pattern predictors. This paper presents a framework for accelerating inference of ReGLU-based feed-forward networks (FFNs) within LLMs. The proposed framework provides a fast, training-free method for building sparse pattern predictors using truncation-aware singular value decomposition (SVD) of the gate projection matrix, along with a threshold calibration algorithm, and inference executors supporting conditional computation on CUDA and CANN devices. Experiments on three sparse LLMs with an average activation sparsity level of 90% in the FFNs demonstrate up to a 1.8x reduction in end-to-end decoding time while maintaining less than 1% degradation in benchmark scores on tasks involving complex math and code generation. This work advances the deployment of LLMs on edge devices.

### 2. 方法论
该框架分为离线校准和在线推理两个阶段：
*   **核心思想**：利用低秩奇异值分解（SVD）近似 FFN 的门控投影矩阵（$W_{gate}$），将其作为轻量级的稀疏模式预测器。
*   **关键技术细节**：
    1.  **截断感知数据白化（Data Whitening）**：在进行 SVD 分解前，利用少量校准数据引入输入统计信息，通过最小化 Frobenius 范数误差来提高低秩近似的精度。
    2.  **校准偏置（Calibrated Bias）**：SVD 近似会导致预测分布偏移。论文引入了神经元特有的偏置项（阈值），并开发了一种**快速贪心算法**，在给定稀疏度预算下，根据神经元对输出精度的贡献度（权重）自动调整阈值。
    3.  **顺序执行流水线（Sequential Pipeline）**：在推理时，先计算门控投影，利用 ReLU 的特性剔除预测中的假阳性（False Positives），进一步提高后续 Up 和 Down 投影的实际稀疏度。
    4.  **硬件执行器**：开发了针对 NVIDIA GPU（CUDA 自定义算子）和华为昇腾（CANN 高阶 API）的稀疏 FFN 执行器。

### 3. 实验设计
*   **实验模型**：选取了三个具有代表性的稀疏化 7B 模型：ProSparse-LLaMA2-7B、SparseQwen2-7B 和 TurboSparse-Mistral-Instruct。

*   **实验规模**：在 3 个模型上进行了 8 个维度的基准测试，并测试了从 0.4 到 0.7 的不同预测稀疏度参数（$s$）。

### 6. 主要结论与发现
*   **加速效果**：在 FFN 层实现约 90% 的激活稀疏度时，端到端（E2E）解码速度提升高达 **1.8 倍**。
*   **精度保持**：在数学和代码等高难度任务中，性能下降控制在 **1% 以内**。
*   **预测能力**：SVD 预测器的 ROC-AUC 得分优于需要训练的 Deja Vu 和 PowerInfer，证明了直接利用模型内部权重结构的优越性。
*   **硬件普适性**：在昇腾硬件上仅通过高层 PyTorch API 即可获得显著加速，证明了该方法对不同架构的友好性。

### 7. 优点
*   **免训练**：极大地降低了部署稀疏 LLM 的门槛，无需昂贵的反向传播训练过程。
*   **理论保障**：为 SVD 近似误差提供了确定性的界限分析（见附录）。
*   **高效性**：通过顺序流水线设计，利用门控结果修正预测错误，实现了比预测稀疏度更高的实际计算稀疏度。
*   **灵活性**：通过调节偏置校准中的稀疏度参数 $s$，可以轻松平衡推理速度与模型精度。

### 8. 不足与局限
*   **架构限制**：目前主要针对 **ReGLU/ReLU** 激活函数的模型。对于使用 SiLU 等平滑激活函数的模型，直接应用会导致精度大幅下降（如 Qwen2-Instruct 实验所示）。
*   **Batch Size 限制**：实验主要针对 **Batch Size = 1** 的场景（边缘设备典型场景）。在高性能服务器的大 Batch 推理中，由于不同序列的稀疏模式难以对齐，加速效果会显著减弱。
*   **Prefill 阶段**：该方法目前仅优化了解码（Decoding）阶段，未解决 Prefill 阶段的稀疏加速问题。
