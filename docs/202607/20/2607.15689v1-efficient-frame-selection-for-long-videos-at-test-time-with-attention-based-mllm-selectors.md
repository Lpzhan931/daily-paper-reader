## 摘要
使用多模态大语言模型（MLLM）理解长视频需要从数千个候选帧中选择一个紧凑的帧集，然而识别正确的帧似乎首先需要理解视频。我们通过一个简单的观察解决了这一循环依赖：在MLLM中经验证选择的提取层上的跨模态注意力已经提供了与查询相关的帧证据，而无需自回归生成。我们利用这一属性构建了DAFS（基于动态注意力的预算感知帧选择器），一种无需训练的帧选择器。即使仅有2B参数的轻量级MLLM选择器，也能通过查询条件聚合将选定层的注意力转换为相关性分数，从而提取帧级证据。这实现了无需自回归解码的跨帧比较。为了处理选择器自身的上下文约束，我们将候选池大小和每帧令牌预算的联合分配形式化为一个由动态规划求解的离散优化问题。在32帧预算下，我们的选择器在Video-MME上相比均匀采样提升了多达6.4个点，并在匹配的帧预算下优于先前基于训练的选择器，同时无需重新训练即可在多种选择器和回答器骨干网络以及任务间泛化。

## Abstract
Understanding long videos with multimodal large language models (MLLMs) requires selecting a compact set of frames from thousands of candidates, yet identifying the right frames seemingly requires understanding the video first. We resolve this circular dependency with a simple observation: cross-modal attention at validation-selected extraction layers in MLLMs already provides query-relevant frame evidence without requiring autoregressive generation. We exploit this property to build DAFS (Dynamic Attention-based Budget-aware Frame Selection), a training-free frame selector. A lightweight MLLM selector, even with only 2B parameters, can extract frame-level evidence by converting selected-layer attention into relevance scores through query-conditioned aggregation. This enables cross-frame comparison without autoregressive decoding. To handle the selector's own context constraint, we formulate the joint allocation of candidate pool size and per-frame token budget as a discrete optimization problem solved by dynamic programming. Under a 32-frame budget, our selector improves over uniform sampling by up to 6.4 points on Video-MME and outperforms prior training-based selectors under matched frame budgets, while generalizing across selector and answerer backbones, and across tasks, without retraining.

- **研究动机**：长视频理解需要从数千帧中选出少量信息帧以匹配多模态大语言模型（MLLM）的有限上下文窗口。但传统方法存在矛盾：均匀采样内容盲目，CLIP 方法缺乏跨帧上下文和强查询条件，MLLM 自回归选择器虽语义深但需额外训练且计算昂贵。

## 2. 论文提出的方法论

- **核心思想**：发现 MLLM 在经验验证的中间层交叉注意力已隐含帧级证据，无需自回归生成。将选定层的文本到视觉注意力转换为帧级相关性分数，实现快速、查询感知的帧选择。
- **关键技术细节**：
  - **注意力提取**：从选定层 ℓ⋆（通过 Charades-STA/NExT-GQA 验证）提取多头注意力均值，得到文本到所有视觉令牌的注意力矩阵 A(ℓ⋆) ∈ Rᵀ×(F·P)。每个帧的原始相关性通过帧内视觉令牌的注意力求和得到 Sₜ,ⱼ。
  - **重要令牌聚合**：对关键查询令牌（如选项令牌或实体令牌）采用最大池化，得到帧显著性分数 ϕⱼ = maxₜ∈T_imp Sₜ,ⱼ，避免非信息令牌的噪声干扰。
  - **预算感知采样与压缩**：候选帧以 1 FPS 提取，再均匀采样 F(d) 帧；每帧通过非重叠平均池化压缩为 P(d) 个令牌（从离散平方数 {16,25,36,49,64,100,144} 中选择）。全局视觉令牌预算 B 固定（如 32,000），满足 F(d)·P(d) ≤ B。
  - **动态规划分配**：将视频时长离散化，在单调性约束（帧数非减、每帧令牌数非增）下，以校准集上的 QA 准确率为目标，通过 DP 求解最优 (F(d), P(d)) 策略。推理时 O(1) 查表。
- **算法流程**：提取候选帧 → 按 DP 策略压缩每帧令牌 → 小 MLLM 前向传播提取选定层注意力 → 重要令牌聚合得到帧分数 → 选 Top-K 帧 → 以原始分辨率送入骨干 MLLM 回答器。整个过程无需训练和自回归解码。

- DAFS 在 32 帧预算下，Video-MME 相对均匀采样提升 6.4 点（Qwen3-VL-2B 选择器），LVB 提升 5.5 点，MLVU 提升 11.5 点。

## 7. 优点

- **免训练**：仅需推理时一次前向，无需额外训练数据或微调，实用性强。
- **轻量高效**：2B 选择器即可获得显著提升，选择阶段仅 0.47 秒，端到端 6.81 秒，优于 ViaRL（12.49 秒）。
- **强查询感知**：通过重要令牌最大池化聚合注意力，聚焦语义关键部分。
- **预算优化**：动态规划联合优化时间覆盖与空间细节，适应不同时长视频。
- **泛化性**：选择器和回答器可互换，无性能退化；跨任务（分类、开放 QA）均有效。

## 8. 不足与局限

- **依赖固定层选择**：层 ℓ⋆ 需在外部数据集上预先验证，若任务分布差异大可能需重新选择。

- **上下文限制仍然存在**：选择器本身的全局视觉令牌预算 B 需手动设定（实验设为 32,000），未讨论自适应调整。
- **未与其他 MLLM 自回归选择器进行全面的端到端成本比较**：仅与 ViaRL 比较，缺少更多同类方法（如 K-Frames, MSJoE 等）的延迟数据。
- **实验仅覆盖英文问答**：未见多语言或细粒度视频理解（如动作检测）评估。
