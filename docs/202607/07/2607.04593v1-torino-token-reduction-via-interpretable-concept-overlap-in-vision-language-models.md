## 摘要
视觉-语言模型（VLM）在各种任务中展现了令人印象深刻的能力，但其计算成本主要由输入到语言模型的大量视觉令牌所主导。现有的令牌缩减方法依赖于基于注意力的分数或成对相似性，而没有对每个令牌进行明确的语义表示。我们提出TORINO（通过可解释概念重叠实现令牌缩减），这是一种即插即用的框架，用于VLM中的自适应视觉令牌缩减，无需对底层模型进行微调。TORINO利用稀疏自编码器（SAE）将视觉令牌投影到可解释的潜在空间中，在该空间中可以通过共享概念激活来分析令牌关系。具体而言，我们将概念重叠定义为活跃SAE潜在变量之间的一致性程度，并利用它来分组共享语义内容的令牌。然后，通过剪枝或合并对每个组内的令牌进行缩减，提供一个统一框架，保留语义重要的视觉信息同时去除冗余。与固定预算方法不同，TORINO根据输入复杂度动态调整缩减率，允许不同图像保留不同数量的令牌。在多个视觉-语言基准上的实验表明，TORINO实现了良好的效率-准确率权衡，以最小的性能损失减少了视觉令牌的数量。

## Abstract
Vision-Language Models (VLMs) have demonstrated impressive capabilities across different tasks, but their computational cost is dominated by the large number of visual tokens fed to the language model. Existing token reduction methods rely on attention-based scores or pairwise similarity, without an explicit semantic representation of each token. We introduce TORINO (TOken Reduction via Interpretable coNcept Overlap), a plug-and-play framework for adaptive visual token reduction in VLMs that requires no fine-tuning of the underlying model. TORINO leverages Sparse Autoencoders (SAEs) to project visual tokens into an interpretable latent space where token relationships can be analyzed through shared concept activations. Specifically, we define concept overlap as the degree of agreement between active SAE latents and use it to group tokens that share semantic content. Reduction within each group is then performed by either pruning or merging, providing a unified framework that preserves semantically important visual information while removing redundancy. Unlike fixed-budget approaches, TORINO dynamically adapts the reduction rate to input complexity, allowing different images to retain different numbers of tokens. Experiments across multiple vision-language benchmarks show that TORINO achieves favorable efficiency-accuracy trade-offs, reducing the number of visual tokens with minimal performance loss.

## 2. 方法论：核心思想、关键技术细节
- **核心思想**：预训练的SAE将每个视觉令牌编码为稀疏概念激活向量，基于令牌间共享的顶级活跃概念的数量构建图，连通分量构成语义组，每组保留一个代表令牌（剪枝或合并），输出令牌数由图像内容自动决定。
- **技术细节**：
  1. **SAE特征提取**：使用Matryoshka BatchTopK SAE（扩展因子ε=64，每令牌活跃特征数K=20），在CLIP ViT-L/14@336的第22层（post-MLP残差流）提取令牌嵌入，得到稀疏概念向量 \( \mathbf{z}_i \in \mathbb{R}^{\omega}_{\geq 0} \)（ω=65536）。同时定义峰值激活分数 \( a(i) = \|\mathbf{z}_i\|_\infty \) 作为概念重要性度量。
  2. **概念引导分组**：对于每个令牌，取top-k活跃概念索引集 \( A_i \)（k ∈ {1,2,3}）。构建无向令牌图，边条件：两个令牌共享至少δ个活跃概念（δ ∈ [1, k]）。连通分量即为语义组 \( \{G_g\} \)。参数对 (k, δ) 控制分组粒度：高 δ/k 产生更多小分组（更保守压缩），低 δ/k 产生更大分组（更激进压缩）。
  3. **令牌缩减**：
     - **TORINO-P（剪枝）**：每组保留峰值激活最高的令牌，丢弃其余令牌。
     - **TORINO-M（合并）**：每组内令牌取对数规模加权平均 \( \mathbf{p}_g = \frac{1+\log|G_g|}{|G_g|} \sum_{i \in G_g} \mathbf{v}_i \)，生成虚拟令牌，补偿大组中冗余信息。
  4. **固定预算变体**：当需要精确B个输出令牌时，添加截断（保留最大B个组）和填充（从候选池中补充高重要性令牌）步骤，保持概念分组原则。

## 7. 优点
- **即插即用**：无需微调VLM，仅需预训练SAE编码器，可集成到现有VLM推理流程。
- **可解释性**：分组基于显式概念（如“天空”、“公牛”、“草地”），而非黑盒相似性，易于理解为何某些令牌被保留或丢弃。
- **内容自适应**：动态输出长度由图像语义复杂度决定，无需显式复杂度估计或预算分配。
- **统一框架**：支持剪枝和合并两种模式，便于研究不同策略的互补性。
- **固定预算变体**：兼顾实际部署中对序列长度一致性的需求，同时保留概念分组核心。
- **效率高**：推理延迟低（~20-30ms），远快于PruneSID/PruMerge，适合实时应用。
- **消融全面**：验证了SAE必要性、扩展因子影响、令牌类型（CLS vs 空间）影响，实验设计严谨。

## 8. 不足与局限
- **场景覆盖有限**：仅评估了LLaVA-1.5系列（7B/13B），未在Qwen-VL、InstructBLIP等其他VLM或不同视觉编码器（如SigLIP）上验证。缺乏多样性。
- **SAE训练局限**：
  - 当前SAE仅使用CLS令牌激活训练（ImageNet-1K），在空间令牌上的分布差异导致性能下降（空间SAE相对分数低5-19pp）。建议训练全补丁令牌SAE。
  - 仅针对CLIP ViT-L/14，其他编码器需重新训练SAE。
- **查询无关性**：当前分组对所有查询同等对待，无法根据问题选择性保留信息。论文提出未来可训练跨模态SAE实现查询条件化。
- **极端压缩下性能下降**：当压缩>90%时，小目标或稀疏概念组易被淘汰，不如基于多样性的方法（如PruneSID）。
- **超参数需手动设置**：(k, δ) 和固定预算B需根据期望压缩水平人工选择，未实现完全自动调整。
- **SAE扩展因子选择依赖经验**：不同压缩级别下最优ε不同（ε=4 vs 64），未提供自适应选择机制。
- **偏差风险**：SAE训练于ImageNet-1K，对域外图像（如医疗、卫星图像）的泛化性未评估。
