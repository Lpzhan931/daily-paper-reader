## 摘要
投机解码已成为加速大语言模型（LLMs）自回归推理的一种极具前景的方法。自草案（Self-draft）方法利用基础 LLM 自身进行投机，避免了辅助草案模型的开销，但仍面临局限性：浅层网络往往会产生过度自信但错误的 token 预测，且草案序列中困难 token 的存在会导致深层网络进行冗余计算，从而降低了草案接受率和整体加速比。为了解决这些问题，我们提出了一种新型自草案框架，通过在早退决策中引入逐层温度退火来抑制虚假置信度，并根据 token 级的解码难度自适应地限制投机长度。通过在深层网络的一次统一并行传递中重新处理草案 token 的隐藏状态，我们的方法在保持与原始模型输出完全等价的同时，最大限度地提升了计算效率。该方法无需修改基础 LLM 参数，在多种长文本生成任务和多种模型架构下，相比标准自回归解码实现了高达 2.33 倍的实际运行时间加速。

## Abstract
Speculative decoding has emerged as a promising approach to accelerate autoregressive inference in large language models (LLMs). Self-draft methods, which leverage the base LLM itself for speculation, avoid the overhead of auxiliary draft models but face limitations: shallow layers often produce overconfident yet incorrect token predictions, and the presence of difficult tokens in a draft sequence forces redundant computation through deeper layers, undermining both draft acceptance and overall speedup. To address these issues, we propose a novel self-draft framework that suppresses spurious confidence via layer-wise temperature annealing in early-exit decision and adaptively bounds speculation length based on token-wise decoding difficulty. By reprocessing the hidden states of draft tokens in a unified parallel pass through deep layers, our method maintains exact output equivalence with the original model while maximizing computational efficiency. It requires no modifications to the base LLM parameters and achieves up to 2.33x wall-time speedup over standard autoregressive decoding across diverse long-form generation tasks and multiple model architectures.

### 1. 核心问题与整体含义（研究动机和背景）
大语言模型（LLM）的自回归生成速度受限于显存带宽。**投机解码（Speculative Decoding）**通过“先草拟再验证”的模式加速推理。现有的“自投机”（Self-draft）方法（利用模型自身浅层进行预测）存在两个核心痛点：
*   **浅层过度自信（Spurious Confidence）：** 预训练目标仅监督顶层输出，导致浅层网络常对错误的预测给出极高的置信度，误导早退（Early-exit）决策。
*   **词元难度异质性（Heterogeneous Difficulty）：** 序列中某些“困难词元”必须经过深层计算才能确定，而投机解码通常采用批处理验证，一个困难词元会拖累整个草案序列进入深层，造成计算冗余。

### 2. 方法论
论文提出了 **SpecBound** 框架，包含两个关键技术：

*   **退火置信度阈值（Annealed Confidence Threshold, ACT）：**
    *   **核心思想：** 在浅层引入更高的采样温度来平滑概率分布，抑制虚假的高置信度峰值。
    *   **技术细节：** 每一层 $\ell$ 的温度 $T_\ell$ 随深度线性递减：$T_\ell = 1 + \alpha(1 - \ell/L)$。浅层温度高（分布平坦），深层回归标准温度（$T=1$）。只有当缩放后的 Top-1 概率超过阈值 $\tau$ 时才允许早退。
*   **带状态缓存的有界投机算法（Bounded Speculation with Cached States, BSCS）：**
    *   **核心思想：** 动态限制投机的深度（$d_{max}$）和宽度（$w_{max}$）。
    *   **流程：** 
        1.  **深度限制：** 若某个词元在达到 $d_{max}$ 层后仍未满足退出条件，则判定为“困难词元”，立即停止投机。
        2.  **宽度限制：** 限制连续投机的最大长度，防止误差累积。
        3.  **并行验证与状态重用：** 利用缓存的隐藏状态，将所有已生成的草案词元在深层（$d_{max}+1$ 到 $L$ 层）进行一次统一的并行前向传递。这确保了输出与原始模型完全一致（无损）。
