## 摘要
在长上下文场景下，预填充（Prefilling）的计算开销构成了大语言模型（LLM）和大模型多模态模型（LMM）的主要瓶颈。虽然 Token 剪枝可以缩短序列长度，但以往的方法依赖于启发式规则，这破坏了与 FlashAttention 等硬件高效算子的兼容性。在这项工作中，我们观察到 Token 会向“语义固定点”（semantic fixing points）演化，使得进一步的处理变得冗余。为此，我们引入了 Delta 注意力选择性停止（DASH），这是一种无需训练的策略，通过监测自注意力机制的逐层更新动态，来选择性地停止已趋于稳定的 Token 的计算。广泛的评估证实，DASH 在语言和视觉基准测试中均具有良好的泛化性，在保持模型准确性和硬件效率的同时，显著提升了预填充速度。代码将发布于 https://github.com/verach3n/DASH.git。

## Abstract
Prefilling computational costs pose a significant bottleneck for Large Language Models (LLMs) and Large Multimodal Models (LMMs) in long-context settings. While token pruning reduces sequence length, prior methods rely on heuristics that break compatibility with hardware-efficient kernels like FlashAttention. In this work, we observe that tokens evolve toward \textit{semantic fixing points}, making further processing redundant. To this end, we introduce Delta Attention Selective Halting (DASH), a training-free policy that monitors the layer-wise update dynamics of the self-attention mechanism to selectively halt stabilized tokens. Extensive evaluation confirms that DASH generalizes across language and vision benchmarks, delivering significant prefill speedups while preserving model accuracy and hardware efficiency. Code will be released at https://github.com/verach3n/DASH.git.

*   **核心问题**：在长上下文场景下，LLM 和 LMM 的预填充阶段（处理输入 Prompt 的过程）计算量随序列长度呈线性或二次方增长，成为推理延迟的主要瓶颈。
*   **研究动机**：
    *   现有的 Token 剪枝方法多依赖复杂的启发式评分，往往需要访问完整的注意力矩阵，这与 FlashAttention 等硬件加速算子不兼容。
    *   作者观察到，Token 在经过若干层处理后会趋向于“语义固定点”，即深层网络的更新量（Delta）变得极小，继续计算这些 Token 产生的边际收益很低。
    *   **核心假设**：稳定性意味着冗余。如果一个 Token 的表示已经稳定，那么它在后续层中可以停止计算。

### 2. 方法论：核心思想与技术细节
*   **核心思想**：通过监测自注意力层的更新幅度（$\Delta_{attn}$）来识别已经“饱和”的 Token，并在中间层将其提前停止（Halting），从而跳过后续层的 Attention 和 FFN 计算。
*   **关键技术细节**：
    1.  **指标定义**：定义 $\Delta_{attn}$ 为自注意力子层在残差相加前的输出向量的 $L_2$ 范数。该指标反映了该 Token 在当前层从全局上下文吸纳新信息的程度。
    2.  **单次选择策略（Single-shot Halting）**：
        *   模型正常处理前 $l_s$ 层。
        *   在指定的起始层 $l_s$，计算所有 Token 的 $\Delta_{attn}$ 分数。
        *   根据分数排序，保留前 $(1-\rho)$ 比例的 Token（Active Set），其余 Token 停止更新。
    3.  **后续处理**：被停止的 Token 在后续层中不再参与计算，其隐藏状态保持在 $l_s$ 层的值。这种方法不改变注意力机制的结构，因此完美兼容 FlashAttention。
*   **多模态差异**：研究发现视觉 Token 比文本 Token 饱和得更早，DASH 能够自动适应这种差异。

### 6. 主要结论与发现
*   **稳定性与冗余正相关**：$\Delta_{attn}$ 较小的 Token 在后续层中被关注的概率也极低，证明其确实是冗余的。
*   **性能优异**：DASH 在 LongBench-E 和 LooGLE 上均取得了比同类剪枝方法更高的平均分。
*   **视觉 Token 冗余度更高**：在多模态模型中，DASH 能够以极高的比例（如 90% 以上）剪枝视觉 Token 而不显著损失精度。
*   **硬件友好**：由于不改变注意力掩码的稀疏结构（只是缩短了序列），在 FlashAttention 下有实际的端到端加速效果。

### 7. 优点
*   **无需训练**：即插即用，适用于现有的预训练模型。
*   **硬件高效**：与现代 GPU 加速内核（FlashAttention）完全兼容，不像某些稀疏注意力方法“理论快、实际慢”。
*   **通用性强**：一套逻辑同时适用于纯文本和多模态输入，且能自动处理不同模态的饱和速度差异。
*   **高压缩比下的鲁棒性**：在极端剪枝（如保留 5% Token）时，性能下降比基准方法更缓慢。
