## 摘要
投机解码（Speculative decoding）加速了大语言模型（LLM）的推理。它使用一个小型的草稿模型来提议未来 token 的树状结构，随后由一个较大的目标模型在单次批处理前向传播中对这些 token 进行验证。尽管关于投机方法的研究日益增多，但任务的认知特征在多大程度上影响接受概率在很大程度上仍未得到探索。我们展示了一项关于基于树的投机解码接受动态的实证研究。我们的研究涵盖了四个成熟的 NLP 基准领域：代码生成、数学推理、逻辑推理和开放式对话。为此，我们使用 TinyLlama-1.1B 作为草稿模型，Llama-2-7B-Chat-GPTQ 作为目标模型。通过从 200 个提示词中收集的 99,768 个投机节点，我们推导出了各领域的接受率、期望接受长度、深度-接受度概况以及熵-接受度相关性。我们发现，任务类型比树深度更能预测接受度。此外，只有对话领域能持续产生每步超过 1.0 个 token 的期望接受长度。我们还表明，在所有领域中，熵与接受度的相关性始终为负但较弱（rho 在 [-0.20, -0.15] 之间）。与直觉相反，对话领域产生了最高的熵，却拥有最高的接受率。我们将这种差异归因于经过 RLHF 对齐的语域（register）在词汇上的可预测性。这些发现对领域感知的投机预算和草稿模型选择策略具有直接意义。索引词：投机解码、大语言模型推理、树注意力、草稿模型、接受概率、LLM 效率。

## Abstract
Speculative decoding accelerates large language model (LLM) inference. It uses a small draft model to propose a tree of future tokens. A larger target model then verifies these tokens in a single batched forward pass. Despite the growing body of work on speculative methods, the degree to which the cognitive characteristics of a task affect acceptance probability remains largely unexplored. We present an empirical study of tree-based speculative decoding acceptance dynamics. Our study spans four well-established NLP benchmark domains: code generation, mathematical reasoning, logical reasoning, and open-ended chat. For this, we use TinyLlama-1.1B as the draft model against Llama-2-7B-Chat-GPTQ as the target. Over 99,768 speculative nodes collected from 200 prompts, we derive per-domain acceptance rates, expected accepted lengths, depth-acceptance profiles, and entropy-acceptance correlations. We find that task type is a stronger predictor of acceptance than tree depth. Furthermore, only the chat domain consistently yields an expected accepted length exceeding 1.0 token per step. We also show that the entropy-acceptance correlation is consistently negative but weak across all domains (rho in [-0.20, -0.15]). Counterintuitively, chat produces the highest entropy yet the highest acceptance rate. We attribute this divergence to the lexical predictability of RLHF-aligned register. These findings have direct implications for domain-aware speculation budgets and draft-model selection strategies. Index Terms--speculative decoding, large language model inference, tree attention, draft model, acceptance probability, LLM efficiency

投机解码（Speculative Decoding）是加速大语言模型（LLM）推理的关键技术，通过小模型（草稿模型）预测、大模型（目标模型）验证来减少延迟。然而，现有研究多关注算法改进，忽略了**任务的认知领域（如数学 vs. 对话）如何影响 token 的接受概率**。
本研究旨在填补这一空白，通过实证分析探讨不同任务类型对投机效率的影响，为“领域感知”的推理优化（如针对不同任务调整投机预算）提供理论依据。

### 2. 论文提出的方法论
研究采用了**基于树的投机解码（Tree-based Speculative Decoding）**框架：
*   **核心思想**：草稿模型不再生成单一序列，而是生成一个候选 token 树，目标模型在单次前向传播中验证整棵树的多条路径。
*   **关键技术细节**：
    *   **树构建**：使用 Top-k 采样和最大分支因子（$b_{max}$）构建深度为 $D_{max}$ 的树。
    *   **接受准则**：采用标准的拒绝采样规则，即接受概率 $\alpha = \min(1, p_{target}/p_{draft})$。
    *   **度量指标**：引入“期望接受长度”（$E[L]$）作为核心效率指标，衡量每次大模型调用平均能产生的 token 数。
    *   **熵分析**：计算目标模型的预测熵（Entropy），以衡量模型的不确定性与接受率之间的相关性。

### 3. 实验设计

    1.  **代码生成**：OpenAI HumanEval（Python 函数补全）。
    2.  **数学推理**：Hendrycks MATH（代数子集）。
    3.  **逻辑推理**：GSM8K（小学数学应用题）。
    4.  **开放式对话**：UltraChat_200k（RLHF 对齐的指令遵循数据）。
*   **实验规模**：每个领域随机抽取 50 个提示词（共 200 个），总计收集了 99,768 个投机节点的观测数据。

### 6. 论文的主要结论与发现
*   **领域分层明显**：**对话（Chat）**是唯一 $E[L] > 1.0$ 的领域（1.065），意味着只有在对话任务中投机解码才产生了实际的加速效果；数学领域表现最差（0.914），投机开销可能超过收益。
*   **深度-接受度正相关**：违背直觉的是，接受率随树深度的增加而略微**上升**。作者将其解释为“上下文承诺效应”：一旦第一层确定了语义方向，后续 token 的语法补全更具确定性。
*   **“对话悖论”**：对话领域拥有最高的预测熵（不确定性高），但同时拥有最高的接受率。这说明 RLHF 训练使模型倾向于使用高度可预测的词汇（如语气词、固定搭配），即使语义空间很广，草稿模型也能轻易猜中目标模型的偏好。
*   **熵与接受率弱相关**：两者呈负相关，但相关系数极弱（$\rho \in [-0.20, -0.15]$），说明熵并非预测投机成功率的可靠指标。

*   **实战指导性强**：提出了“请求路由”建议，即根据任务类型动态开启/关闭投机解码，或为数学任务选择专门的草稿模型。
