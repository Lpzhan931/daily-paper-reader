## 摘要
投机性解码 (SD) 通过使用较小的模型来起草未来的 token，并由目标大语言模型 (LLM) 进行验证，从而加速 LLM 的推理。这种方式通过仅接受对齐的 token 来保证生成质量。然而，单个草稿模型通常针对特定任务或领域进行训练，在多样化的应用中有效性有限。为解决这一问题，我们引入了 MetaSD，这是一个将多个草稿模型集成到 SD 过程中的统一框架。MetaSD 通过利用对齐反馈并将草稿模型选择建模为多臂老虎机 (multi-armed bandit) 问题，动态地为异构草稿模型分配计算资源。大量实验表明，MetaSD 始终优于单草稿模型方法。

## Abstract
Speculative decoding (SD) accelerates large language model (LLM) inference by using a smaller model to draft future tokens, which are then verified by the target LLM. This preserves generation quality by accepting only aligned tokens. However, individual drafters, often trained for specific tasks or domains, exhibit limited effectiveness across diverse applications. To address this, we introduce \textsc{MetaSD}, a unified framework that integrates multiple drafters into the SD process. MetaSD dynamically allocates computational resources to heterogeneous drafters by leveraging alignment feedback and framing drafter selection as a multi-armed bandit problem. Extensive experiments show MetaSD consistently outperforms single-drafter approaches.

*   **核心问题**：传统的投机性解码（Speculative Decoding, SD）依赖单一的小型草拟模型来预测后续 token。然而，单一草拟器通常在特定领域（如代码或翻译）表现良好，但在面对多样化、动态的实际用户查询时，其预测准确率（对齐度）会大幅下降，导致加速效果受限甚至失效。
*   **研究背景**：随着专用小模型（Specialized Models）的增多，如何从多个异构草拟器中动态选择最匹配当前上下文的一个，成为了提升 SD 鲁棒性和泛化性的关键。

### 2. 论文提出的方法论
*   **核心思想**：将多草拟器的选择问题建模为**多臂老虎机（Multi-Armed Bandit, MAB）**问题。在推理的每一个循环中，系统根据历史表现动态选择一个草拟器，并利用大模型的验证结果作为反馈来更新选择策略。
*   **关键技术细节**：
    *   **对齐反馈（Alignment Feedback）**：引入**块散度（Block Divergence, BD）**作为奖励信号。相比于传统的块效率（BE，仅计算接受 token 数量），BD 利用目标模型与草拟器之间的总变差距离（TV Distance）来衡量分布对齐度，提供更丰富、方差更小的学习信号。
    *   **MetaSD-UCB 算法**：采用置信区间上界（UCB）算法平衡“探索”（尝试不同草拟器）与“利用”（使用当前表现最好的草拟器）。
    *   **理论保障**：论文证明了 MetaSD-UCB 在目标序列长度 $B$ 下具有对数级的遗憾上界（Regret Bound），并证明了 BD 奖励比 BE 奖励能更快收敛。
    *   **框架兼容性**：支持“黑盒”（独立小模型）和“白盒”（如 Eagle 等共享隐藏层状态的模型）两种配置。

### 6. 主要结论与发现
*   **性能提升**：MetaSD-UCB 在所有任务中均显著优于单一草拟器和静态路由方法。在代码和数学任务中，加速比相比基准有大幅提升。
*   **奖励机制优势**：实验证实 BD 奖励在识别最优草拟器方面比 BE 奖励更准确、更快速（Best Arm Ratio 更高）。
*   **鲁棒性**：MetaSD 对提示词的微小改动不敏感，且能有效处理非平稳环境（即一个 query 中包含多个子任务的情况）。
*   **低开销**：虽然加载了多个草拟器，但由于草拟器极小（68M），显存占用仅增加约 2GB，且每次仅激活一个草拟器，不增加计算带宽负担。

### 7. 优点（亮点）
*   **无需重新训练**：MetaSD 是一个“即插即用”的推理框架，可以整合现有的、针对不同任务训练好的异构草拟器。
*   **理论与实践结合**：不仅提供了显著的推理加速，还给出了严谨的老虎机遗憾上界证明。
*   **动态适应性**：解决了 SD 在多任务混合场景下的性能退化问题，使投机解码更具实用价值。

### 8. 不足与局限
*   **显存占用**：虽然单个草拟器很小，但如果草拟器池规模极大（例如成百上千个），显存压力和管理复杂度会显著增加。
