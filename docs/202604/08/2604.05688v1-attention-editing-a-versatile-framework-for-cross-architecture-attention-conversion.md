## 摘要
在长上下文和长生成场景下，键值（KV）缓存的显存占用和带宽需求日益成为大语言模型推理成本的主导因素。诸如多头潜在注意力（MLA）和混合滑动窗口注意力（SWA）等架构可以缓解这一限制，但将它们集成到现有模型中仍然具有挑战性。先前的方法对源注意力模块和目标注意力模块都有细粒度的结构要求，无法满足实际部署中的可行性需求。我们提出了 Attention Editing，这是一个实用的框架，用于在无需从头开始重新预训练的情况下，将已训练的大语言模型（LLM）转换为新的注意力架构。Attention Editing 用可学习的目标模块替换原始注意力模块，并使用渐进式蒸馏进行训练，该过程包括：（1）带有中间激活监督的分层教师强制（teacher-forced）优化，以防止冷启动误差累积；（2）针对下一标记（next-token）分布的模型级蒸馏，并可选择性地通过弱特征匹配进行正则化。我们在两种不同的目标架构——MLA 和 GateSWA（一种门控混合 SWA 设计）上实例化了该框架，并将其应用于 Qwen3-8B 和 Qwen3-30B-A3B。所得模型在保持竞争性能的同时，实现了显著的效率提升，证明了大规模注意力转换既可行又稳健。值得注意的是，实验是在昇腾（Ascend）910B 集群上进行的，为国产硬件上的训练提供了实际案例研究。

## Abstract
Key-Value (KV) cache memory and bandwidth increasingly dominate large language model inference cost in long-context and long-generation regimes. Architectures such as multi-head latent attention (MLA) and hybrid sliding-window attention (SWA) can alleviate this bound, but integrating them into existing models remains difficult. Prior methods impose fine-grained structural requirements on both source and target attention modules, which cannot meet the feasible requirement in practical deployment. We present Attention Editing, a practical framework for converting already-trained large language models (LLMs) with new attention architectures without re-pretraining from scratch. Attention editing replaces the original attention with a learnable target module and trains it using progressive distillation, consisting of (1) layer-wise teacher-forced optimization with intermediate activation supervision to prevent cold-start error accumulation, and (2) model-level distillation on next-token distributions, optionally regularized by weak feature matching. We instantiate the framework on two different target--MLA and GateSWA, a gated hybrid SWA design, and apply it to Qwen3-8B and Qwen3-30B-A3B. The resulting models maintain competitive performance while delivering substantial efficiency improvements, demonstrating that large-scale attention conversion is both feasible and robust. Notably, experiments are conducted on an Ascend 910B clusters, offering a practical training case study on domestic hardware.

### 1. 核心问题与研究动机
*   **核心问题**：在长上下文和长生成任务中，大模型的推理成本主要受限于 **KV 缓存（KV Cache）** 的显存占用和带宽。
*   **研究背景**：虽然 MLA（多头潜在注意力）和 SWA（滑动窗口注意力）等架构能显著降低 KV 缓存，但将现有已训练好的模型（如基于 GQA 的模型）转换为这些架构通常需要从头预训练，成本极高。
*   **现有局限**：先前的转换方法（如 TransMLA）依赖于精细的权重手术（如 SVD 分解）和特定的位置编码处理，对源模型和目标模型的结构匹配要求过于苛刻，难以应用于复杂的生产环境或经过后训练（SFT/RL）的模型。

### 2. 方法论：Attention Editing 框架
该框架的核心思想是**将目标注意力模块视为可学习的替代品**，通过“渐进式蒸馏”实现架构转换，而不依赖复杂的权重初始化。

*   **核心流程**：
    1.  **模块替换**：将原始注意力模块替换为新架构（如 MLA 或 GateSWA），新模块的参数除输出投影层（o_proj）外，大多采用随机初始化。
    2.  **第一阶段：层级教师强制蒸馏（Layer-wise Teacher-forced Distillation）**：
        *   为了防止冷启动时的误差在深层网络中累积，该阶段将每一层独立训练。
        *   输入使用教师模型（原模型）的隐藏状态，监督信号是教师模型注意力分支的输出（post-o_proj 激活值），使用 MSE 损失函数进行对齐。
    3.  **第二阶段：模型级分布蒸馏（Model-level Distillation）**：
        *   在模块获得初步能力后，进行端到端训练。
        *   通过 KL 散度匹配教师和学生模型的下一标记（Next-token）输出分布。
        *   **正则化**：引入弱中间特征匹配（余弦相似度损失）以稳定训练。
*   **创新架构 GateSWA**：提出了一种带门控的滑动窗口注意力，通过元素级门控（Gate）替代传统的 Sink Token 机制，有效缓解了注意力汇聚（Attention Sink）问题并提升了稳定性。

*   **模型对象**：Qwen3-8B（稠密模型）和 Qwen3-30B-A3B（MoE 模型）。

*   **硬件平台**：全量训练均在 **华为昇腾（Ascend）910B 集群**上完成，展示了国产算力在大模型架构演进中的实用性。

*   **推理测试**：在 H800 GPU（使用 vLLM）和 Ascend 910B 上分别进行了效率基准测试。

### 5. 实验充分性与客观性
*   **实验规模**：涵盖了不同规模（8B/30B）和不同类型（Dense/MoE）的模型，验证了框架的普适性。
*   **对比维度**：不仅关注模型精度（Accuracy），还深入分析了 KV 缓存减少比例和实际推理速度提升，实验维度较为全面。
*   **客观性**：作者承认了由于无法获取 Qwen3 原始预训练私有数据，实验使用了开源数据集替代，这反而证明了该方法对数据分布变化的鲁棒性。

### 6. 主要结论与发现
*   **转换可行性**：即使从随机初始化开始，通过渐进式蒸馏也能使新架构恢复原模型绝大部分的对话和推理能力。
*   **效率提升显著**：GateSWA 将 KV 缓存降低了约 80%；在并发增加时，转换后的模型吞吐量优势随之扩大。
*   **性能保持**：在 GSM8K 等任务上，转换后的模型性能与原模型非常接近，证明了注意力模块主要承担知识检索功能，而知识存储（FFN 层）在转换中得到了保留。

### 7. 优点与亮点
*   **通用性强**：不要求源/目标架构之间存在严格的数学对应关系，支持跨架构的大幅修改。
*   **数据高效**：仅需极少量数据（<10B Tokens）即可完成复杂架构的迁移。
*   **实战价值**：提供了在国产昇腾硬件上进行大规模模型编辑的完整技术路线。
*   **GateSWA 设计**：简化了滑动窗口注意力的实现，通过门控机制提升了长文本处理的稳定性。

### 8. 不足与局限
*   **知识损耗**：在知识密集型任务（如 C-Eval）中，转换后的模型相比原模型仍存在一定的性能差距。
*   **数据规模上限未知**：目前仅使用了 8B Tokens，尚未探索增加数据量是否能完全抹平性能差距。
*   **任务覆盖**：实验主要集中在通用对话和数学推理，对于 Agent 场景（如工具调用、长程规划）的评估相对较少。
