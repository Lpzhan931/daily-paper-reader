## 摘要
标准注意力机制的平方级计算复杂度构成了大语言模型在长上下文推理中的根本瓶颈。虽然现有的 KV 缓存压缩方法缓解了内存压力，但它们往往会牺牲生成质量，且未能解决浮点运算的高开销问题。本文介绍了 DASH-KV，这是一种创新的加速框架，它通过非对称深度哈希将注意力机制重新表述为近似最近邻搜索。在此范式下，我们设计了一种非对称编码架构，对查询（queries）和键（keys）进行差异化映射，以考虑它们在精度和重用特性上的区别。为了平衡效率和准确性，我们进一步引入了一种动态混合精度机制，自适应地为关键标记（tokens）保留全精度计算。在 LongBench 上的广泛实验表明，DASH-KV 在显著优于现有最先进基准方法的同时，达到了与全注意力机制相当的性能，并将推理复杂度从 O(N^2) 降低到线性 O(N)。代码可在 https://github.com/Zhihan-Zh/DASH-KV 获取。

## Abstract
The quadratic computational complexity of the standard attention mechanism constitutes a fundamental bottleneck for large language models in long-context inference. While existing KV cache compression methods alleviate memory pressure, they often sacrifice generation quality and fail to address the high overhead of floating-point arithmetic. This paper introduces DASH-KV, an innovative acceleration framework that reformulates attention as approximate nearest-neighbor search via asymmetric deep hashing. Under this paradigm, we design an asymmetric encoding architecture that differentially maps queries and keys to account for their distinctions in precision and reuse characteristics. To balance efficiency and accuracy, we further introduce a dynamic mixed-precision mechanism that adaptively retains full-precision computation for critical tokens. Extensive experiments on LongBench demonstrate that DASH-KV significantly outperforms state-of-the-art baseline methods while matching the performance of full attention, all while reducing inference complexity from O(N^2) to linear O(N). The code is available at https://github.com/Zhihan-Zh/DASH-KV

### 2. 方法论：核心思想与关键技术
DASH-KV 框架的核心思想是利用**非对称深度哈希**构建语义哈希空间，将注意力得分的计算转化为汉明距离（Hamming Distance）的匹配。

*   **非对称哈希架构**：
    *   **查询（Query）编码**：使用轻量级多层感知机（MLP）动态捕捉复杂的语义信息，确保检索精度。
    *   **键（Key）编码**：采用简单的线性投影，以适应 Key 在缓存中的高频重用和高效存储需求。
*   **多维校准策略**：
    *   **跨头共识（Cross-Head Consensus）**：利用多头注意力的空间先验，通过“投票机制”修正单个头的判断偏差。
    *   **跨层动量（Cross-Layer Momentum）**：利用深层网络的时序依赖，参考前一层的注意力分布作为先验。
*   **动态混合精度注意力**：
    *   根据校准后的汉明距离将 Token 分为三类：**高相关**（保留全精度计算）、**中相关**（采用“哈希+残差补偿”方案）、**弱相关**（直接掩码剔除）。

### 6. 主要结论与发现
*   **性能匹配**：DASH-KV 在多个任务上达到了与全量注意力（Full Attention）相当的水平，显著优于 StreamingLLM 和 H2O 等基准。
*   **复杂度降低**：成功将注意力机制的计算复杂度从 $O(N^2)$ 降低到线性 $O(N)$。
*   **层敏感性**：模型的前几层和最后几层对精度极度敏感，而中间层具有较大的冗余空间，适合应用哈希加速（即“三明治部署策略”）。
*   **非对称优势**：证明了 Query 和 Key 在数值分布上存在显著差异，非对称哈希比对称哈希更能有效保留语义特征。

### 7. 优点与亮点
*   **范式创新**：首次系统性地将深度哈希检索技术引入 Transformer 内部注意力优化，打破了传统的浮点运算框架。
*   **非对称设计**：针对 Q 和 K 的不同特性量身定制编码器，平衡了计算开销与表征精度。
*   **动态自适应**：通过混合精度机制和百分比阈值，实现了实例级的效率-准确率权衡，避免了固定剔除带来的信息永久丢失。

### 8. 不足与局限
*   **硬件算子支持**：目前主要在深度学习框架内模拟哈希运算，尚未开发专门针对低位宽位运算优化的 GPU 内核（如自定义 CUDA Kernel），实际加速比受限于现有框架的 I/O。
*   **层依赖训练**：由于各层分布差异大，该方法需要针对不同层进行独立参数优化（Layer-Wise Training），增加了部署前的离线处理成本。
*   **极端场景风险**：虽然在 32k 长度下表现良好，但在处理超长文本（如 128k+）时，哈希冲突对精度的影响仍需进一步验证。
