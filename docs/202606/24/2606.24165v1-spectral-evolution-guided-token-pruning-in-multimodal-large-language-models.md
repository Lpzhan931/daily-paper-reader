## 摘要
减少视觉令牌冗余对于加速多模态大语言模型（MLLMs）且不降低跨模态推理性能至关重要。现有的令牌剪枝方法通常依赖单层信号，如注意力分数或令牌相似度，这忽略了视觉表示的跨层变换，并可能在多模态令牌序列中表现出位置偏差。为解决这一局限，我们提出了一种基于跨层频谱演化（CLSE）的无训练令牌剪枝框架。CLSE不是从单层特征幅度衡量令牌重要性，而是在频域中量化令牌表示在Transformer层间的演化。这种演化反映了从高频结构细节到低频语义抽象的转变。我们观察到，跨层具有更强频谱重分布的令牌更可能是语义活跃的，因此应被保留。通过建模跨层令牌动态，CLSE提供了稳定的重要性标准，减轻了位置偏差。在图像和视频基准上的大量实验表明，CLSE在激进令牌缩减下实现了效率与准确性的优越权衡。在多种MLLMs上，CLSE减少了FLOPs、KV缓存内存和延迟，同时保持了竞争力或更优的性能。

## Abstract
Reducing visual token redundancy is critical for accelerating Multimodal Large Language Models (MLLMs) without degrading cross-modal reasoning performance. Existing token pruning methods typically rely on single-layer signals, such as attention scores or token similarities, which overlook the cross-layer transformation of visual representations and may exhibit positional bias in multimodal token sequences. To address this limitation, we propose a training-free token pruning framework based on Cross-Layer Spectral Evolution (CLSE). Instead of measuring token importance from single-layer feature magnitudes, CLSE quantifies how token representations evolve across Transformer layers in the frequency domain. This evolution reflects the transition from high-frequency structural details to low-frequency semantic abstractions. We observe that tokens with stronger spectral redistribution across layers are more likely to be semantically active and should therefore be preserved. By modeling cross-layer token dynamics, CLSE provides a stable importance criterion that mitigates positional bias. Extensive experiments on both image and video benchmarks demonstrate that CLSE achieves a superior trade-off between efficiency and accuracy under aggressive token reduction. Across multiple MLLMs, CLSE reduces FLOPs, KV cache memory, and latency while maintaining competitive or improved performance.

- **研究动机**：观察发现视觉令牌在LLM深层中会从“高频结构细节”逐步转化为“低频语义抽象”，而语义活跃的令牌具有更强的跨层频谱重分布，背景令牌则相对静态。因此，通过建模跨层频谱演化可提供更可靠的令牌重要性信号。
- **整体含义**：提出一种训练自由的令牌剪枝框架，在频域中量化令牌的跨层演化强度，保留语义活跃令牌，实现高效推理而不损害多模态推理能力。

## 2. 论文提出的方法论：核心思想、关键技术细节
- **核心思想**：跨层频谱演化（Cross-Layer Spectral Evolution, CLSE）。通过FFT将每层视觉令牌特征图映射到频域，应用**高斯高通滤波器**抑制低频全局模式，再逆变换回空间域得到高频残余能量图。测量相邻层间残余能量图的相对变化，作为令牌的重要性分数。
- **关键技术细节**：
  1. **FFT与高通滤波**：对视觉特征图\(X_{\ell}^{2D}\)进行2D FFT，频谱中心化后乘以高斯高通掩膜\(M\)（截止比\(r\)控制），再逆变换获得残余图\(R_\ell\)，平均通道得到每令牌高频能量分布\(S_\ell\)。
  2. **演化强度计算**：定义\(I^{(\ell)} = \text{clip}\left(\frac{|S_{\ell+1} - S_\ell|}{S_\ell + \text{mean}(S_\ell) + \epsilon}, 0, \tau_{\max}\right)\)，相对度量确保尺度不变，全局均值正则化抑制低能量区域。
  3. **剪枝策略**：默认以LLM解码器第0→1层作为参考，在第1层之后剪枝。每样本根据演化强度选取Top-K令牌保留，后续层只处理缩减序列。
  4. **理论分析**：证明在固定预算下，选取Top-K演化强度的令牌可最小化剪枝导致的下游扰动上界，保留最大化的跨层频谱演化质量。
- **可扩展性**：与令牌合并方法（如ToMe）正交，可结合使用（CLSE-M），在视频任务中通过回收高演化令牌并聚类压缩实现更优效果。

- **位置偏差缓解**：定性分析显示CLSE聚焦语义目标区域，而注意力受位置编码干扰。

## 7. 优点：方法或实验设计上的亮点
- **训练自由、即插即用**：无需修改模型或重新训练，可直接集成到现有MLLM推理管线。
- **跨层频谱动态建模**：利用FFT高效计算令牌演化，复杂度近似线性，优于注意力回顾（O(N²)）。
- **理论保证**：证明Top-K剪枝最小化下游扰动上界，提供严谨数学解释。
- **消融实验全面**：从截止比、组件贡献、跨层度量、剪枝层位置等多角度验证设计合理性。
- **多模态泛化**：同时覆盖图像和视频任务，并分别设计了2D FFT和CLSE-M合并变体。
- **大规模验证**：在72B参数模型上仍有效，展示可扩展性。

## 8. 不足与局限
- **仅加速预填充阶段**：当前方案主要减少预填充令牌，对自回归生成阶段的加速（如KV缓存压缩）需额外工作。
- **未评估细粒度幻觉**：尽管POPE指标评估幻觉，但未系统评估生成文本中的事实错误或幻觉率。
- **架构覆盖有限**：虽然包括Qwen2-VL和InternVL，但未在更多LLaVA变体（如LLaVA-HR）或CogVLM上验证。
- **高通滤波参数敏感**：截止比\(r\)需手动设置（默认0.1），不同图像或任务可能需调整。
- **视频中时间维度利用不足**：仅使用稀疏帧的2D FFT，未充分建模帧间光流或时序频谱，可能损失跨帧动态。
- **未考虑训练剪枝联合优化**：作为训练自由方法，无法通过任务损失反向传播调整重要性标准，可能不如端到端方法。
