## 摘要
自回归模型通过将图像视为离散标记的一维序列，在视觉生成中表现出色，类似于语言建模。然而，这种展平处理丢弃了视觉信号固有的二维空间局部性，在推理过程中造成严重的计算瓶颈。我们提出了空间推测解码（SSD），这是一个将预测目标与图像自然几何结构对齐的框架。我们的模型不是仅预测一维序列中的下一个直接标记，而是同时预测相邻的水平标记和正下方的标记。通过利用这种二维空间相关性，空间推测解码克服了视觉推理中的内存墙。我们的方法在保持DPG-Bench和GenEval高保真度的同时，将自回归图像生成加速高达13.3倍。我们的结果表明，尊重视觉的底层几何结构可以释放巨大的计算效率，为实时、高分辨率自回归生成模型铺平道路。

## Abstract
Autoregressive models excel in visual generation by treating images as 1D sequences of discrete tokens, mirroring language modeling. However, this flattening discards the intrinsic 2D spatial locality of visual signals, creating severe computational bottlenecks during inference. We introduce Spatially Speculative Decoding (SSD), a framework that aligns the predictive objective with the natural geometry of images. Rather than predicting only the immediate next token in a 1D sequence, our model simultaneously predicts the adjacent horizontal token and the token directly below it. By capitalizing on this 2D spatial correlation, spatially speculative decoding overcomes the memory wall in visual inference. Our approach accelerates autoregressive image generation by up to 13.3x while maintaining high fidelity on DPG-Bench and GenEval. Our results suggest that respecting the underlying geometry of vision unlocks massive computational efficiencies, paving the way for real-time, high-resolution autoregressive generative models.

## 1. 核心问题与整体含义（研究动机和背景）

自回归视觉模型将图像展平为一维 token 序列，借鉴语言建模的“下一个 token 预测”范式，实现了视觉与语言的统一。然而，这种展平操作丢弃了图像固有的二维空间局部性，导致推理时存在严重的内存墙瓶颈：生成一张高分辨率图像需要数千次串行前向传播（O(n²) 复杂度），每次加载全部模型参数却只产生一个 token。现有加速方法（如 1D 推测解码、Jacobian 迭代）仍局限于 1D 序列假设，在视觉任务中仅获得 1.8×–3.7× 的有限加速，且常伴随质量下降。论文提出**空间推测解码（SSD）**，核心洞察是视觉空间相关性本质上是二维的：垂直方向上的相邻 token 与水平方向的紧邻 token 具有相当的预测可靠性。通过同时沿水平和垂直方向预测 token，SSD 将推理复杂度从 O(n²) 降低至 O(n)，实现高达 13.3× 的实际墙钟加速，同时保持高生成保真度。

## 2. 提出的方法论：核心思想与关键技术细节

- **核心思想**：将二维空间预测分解为两个正交的一维预测流——水平方向（沿光栅扫描顺序）和垂直方向（利用已完成的行直接预测整行 token）。通过训练轻量级预测头，在连续隐空间（最后一层 Transformer 的预归一化隐藏状态）中同时预测相邻 token。
- **关键技术细节**：
  - **预测目标与输入**：预测最后一层 Transformer 隐藏状态（`h_pre`）前经过 RMSNorm 的特征，并拼接 token 嵌入 `e_y,x` 以消除采样不确定性。预测头使用单层 SwiGLU 网络，对每个偏移 δ 独立训练。
  - **二维预测分解**：设置少量水平头（例如 +1, +2, +3）和垂直头（例如 +48, +96, +144），利用垂直方向列间独立性，在一个并行步骤中完成整行的预测。
  - **验证即自动修正（Verification as Auto Correction）**：不是严格拒绝所有不匹配 token，而是通过一次前向传播计算目标分布，对每个 draft token 独立执行标准推测接受规则。拒绝的 token 从残差分布重新采样，并回滚 KV 缓存后重新验证（共 r 轮）。这避免了回滚整块 draft，允许大规模空间块的一次性验证。
  - **算法流程**（文字说明）：
    1. 使用水平头沿当前行 draft 至行尾（O(1) 次前向传播）。
    2. 使用垂直头同时 draft 后续多行的所有 token（整行并行）。
    3. 将整块 draft 输入骨干网络并行验证，执行 r 轮“验证+修正”，每轮只增加一次前向传播。
    4. 最终一次前向传播提交验证后的块并更新 KV 缓存。
  - **无模型修改**：SSD 仅添加轻量级预测头，不改变预训练骨干网络，可作为即插即用模块。

## 6. 主要结论与发现

- SSD 在 Lumina-mGPT 上实现 12.19× 墙钟加速，在 Janus-Pro 上 5.70×，在 Emu3 上 13.28×，同时质量损失极小（DPG-Bench 分数变化在 ±3 分以内）。

- 分阶段验证（先提交前导行再修正后续行）比联合验证更稳定、质量更高。
