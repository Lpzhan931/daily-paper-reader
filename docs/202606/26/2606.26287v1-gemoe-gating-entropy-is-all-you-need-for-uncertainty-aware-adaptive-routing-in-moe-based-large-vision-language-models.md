## 摘要
随着模型参数和训练数据的增加，大型视觉语言模型（LVLMs）的指令遵循和泛化能力得到了显著提升。基于混合专家（MoE）架构，LVLMs在保持推理成本的同时扩大了参数容量。然而，传统的MoE方法采用Top-k静态路由策略，未能考虑输入的变化并自适应地选择专家数量，导致资源利用欠佳。本文提出将令牌路由视为一项信息编码任务，将动态路由框架视为编码中的最小描述长度（MDL）问题。通过在MoE场景中验证MDL与门控熵之间的联系，我们引入了基于门控熵的不确定性感知自适应路由（GeMoE）用于MoE。与传统的静态或基于启发式的动态路由方法不同，GeMoE显式地对模型复杂度与性能之间的权衡进行建模。通过使用门控熵评估令牌的复杂度，GeMoE自适应地决定每个令牌应参与的专家数量。在广泛的骨干网络和基准测试中，与原始静态路由相比，我们的方法平均性能保持率达到99.5%，同时平均专家激活稀疏度提高了36.5%。

## Abstract
With the increase in model parameters and training data, the instruction following and generalization capabilities of Large VisionLanguage Models (LVLMs) have been significantly improved. Based on the Mixture of Experts (MoE) architecture, LVLMs expand their parameter capacity while maintaining the inference cost. However, traditional MoE methods employ a Top-k static routing strategy, which fails to account for variations in the input and adaptively select the number of experts, resulting in suboptimal resource utilization. In this paper, we propose viewing token routing as an information encoding task, framing dynamic routing as a Minimum Description Length (MDL) problem in encoding By validating the connection between MDL and gating entropy in the MoE scenario, we introduce Gating Entropy-based Uncertainty-aware Adaptive Routing (GeMoE) for MoE. Unlike traditional static or heuristic-based dynamic routing methods, GeMoE explicitly models the trade-off between model complexity and performance. By using gating entropy to assess the complexity of tokens, GeMoE adaptively determines the number of experts each token should engage. On a wide range of backbones and benchmarks, our method achieves 99.5% average performance retention compared to the original static routing, while improving average expert activation sparsity by 36.5%.

## 1. 核心问题与整体含义（研究动机和背景）
- **研究背景**：大型视觉语言模型（LVLMs）通过混合专家（MoE）架构在不显著增加推理成本的情况下扩大参数容量，但传统MoE采用**Top-k静态路由**，为每个输入固定激活相同数量的专家。
- **核心问题**：静态路由忽略了不同令牌（token）的信息复杂度差异，导致低信息量令牌浪费计算资源，高信息量令牌专家不足，性能受损。现有动态路由方法（如阈值法、伪专家法）基于启发式策略，未能**显式地建模模型复杂度与性能之间的权衡**。
- **整体含义**：该问题本质上是一个**最小描述长度（MDL）最优编码问题**——需要自适应地为不同复杂度的令牌分配不同数量的专家（编码长度），从而在保持性能的同时提高专家利用率。本文通过建立MDL与门控熵的联系，实现了高效、可解释的动态路由。

## 2. 方法论：核心思想、关键技术细节
### 核心思想
- 将令牌路由视为**信息编码任务**：专家集合视为码本，静态Top-k是固定长度编码，动态路由是可变长度编码，目标是**最小化MDL**（模型复杂度编码长度 + 数据拟合编码长度）。
- 证明**门控熵（Gating Entropy）可以作为MDL的代理**：当令牌在门控分布上熵较高（路由不确定性大）时，添加额外专家能带来较大的信息增益（Info Gain），从而降低MDL；反之，低熵令牌添加专家增益小。（见公式7-8及图3验证）
- 基于此，设计**分层学习机制**：先根据门控熵确定令牌需要的专家数量（正相关），再通过轻量级预测模块输出具体的专家数量。

### 关键技术细节
1. **门控熵计算**：对令牌 \(x\)，利用路由器输出的分布 \(R_{\text{norm}}(x)\) 计算香农熵：  
   \( H(x) = -\sum_{i=1}^{K} w_i(x) \log_2 w_i(x) \)。
2. **专家数量预测器（Expert Assignment Predictor, EAP）**：一个轻量线性层，输入令牌特征，输出每个可能专家数量（1~8）的概率分布，通过期望值 \(k_{\text{soft}}\) 和取整得到最终激活数量 \(k\)。
3. **单调损失（Monotonic Loss）**：确保门控熵高的令牌被分配更多专家，比低熵令牌的预测值大一个间隔。  
   \( \mathcal{L}_{\text{mono}} = \sum_{(x_i,x_j)} \max(0, \text{margin} - k_{\text{soft}}^i + k_{\text{soft}}^j) \) 满足条件时计算。
4. **整体损失**：联合交叉熵损失（\(\mathcal{L}_{\text{ce}}\)）、单调损失（\(\mathcal{L}_{\text{mono}}\)）和负载均衡损失（\(\mathcal{L}_{\text{lb}}\)），权重超参数 \(\alpha=1.0, \beta=0.001\)。
5. **推理流程**：每个令牌通过路由器获得门控分布 → EAP预测 \(k\) → 激活Top-k专家并加权组合输出。

### 算法描述（文字）
- 对每个批次令牌，计算标准化路由分数，获得门控熵。
- 通过EAP预测软专家数量 \(k_{\text{soft}}\)，四舍五入得到整数 \(k\)。
- 仅激活分数最高的 \(k\) 个专家，计算输出。
- 训练时额外计算单调损失，反向传播优化EAP和路由器（专家本身冻结）。
