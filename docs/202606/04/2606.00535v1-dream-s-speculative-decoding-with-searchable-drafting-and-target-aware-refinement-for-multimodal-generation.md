## 摘要
推测解码（SD）已被证明是加速大型语言模型（LLMs）自回归生成的有效技术，然而，其在视觉-语言模型（VLMs）上的应用仍相对未被探索。我们提出了~\textit{DREAM-S}，一种专为VLMs中快速高效解码而设计的新型SD框架。DREAM-S利用神经架构搜索（NAS）框架与目标感知超网络训练，自动识别草稿模型与目标模型之间的最优交互策略，以及最适合底层硬件实现平台的草稿模型架构。此外，DREAM-S还结合了由注意力熵引导的自适应中间特征蒸馏，以实现高效的草稿训练。在一系列成熟的VLMs上的实验表明，与标准解码方法相比，DREAM-S实现了高达$3.85\times$的加速，并显著优于现有的SD基线。代码公开于：https://github.com/SAI-Lab-NYU/DREAM-S 。

## Abstract
Speculative decoding (SD) has proven to be an effective technique for accelerating autoregressive generation in large language models (LLMs) however, its application to vision-language models (VLMs) remains relatively unexplored. We propose~\textit{DREAM-S}, a novel SD framework designed specifically for fast and efficient decoding in VLMs. DREAM-S leverages a neural architecture search (NAS) framework with target-aware supernet training to automatically identify both the optimal interaction strategy between the draft and target models, and the most suitable draft model architecture for the underlying hardware implementation platform. DREAM-S additionally incorporates adaptive intermediate feature distillation, guided by attention entropy, to enable efficient draft training. Experiments on a range of well-established VLMs show that DREAM-S achieves up to a $3.85\times$ speedup compared to standard decoding approaches and significantly outperforms existing SD baselines. The code is publicly available at: https://github.com/SAI-Lab-NYU/DREAM-S .

## 2. 论文提出的方法论

- **核心思想**：将 NAS 引入 VLM 推测解码的草稿模型设计，自动搜索三个关键维度（注意力头剪枝比率、视觉 token 压缩比率、目标特征注入层位置），并采用两阶段渐进训练（Two-Phase Progressive Training, TPPT）和自适应中间特征蒸馏（Adaptive Intermediate Feature Distillation, AIFD）来训练草稿模型。
- **关键技术细节**：
  - **注意力头剪枝**：基于梯度和权重大小的乘积计算每个注意力头的重要性，在训练时动态选择保留顶部 H' 个头。
  - **视觉 token 压缩**：利用目标模型最后一层的注意力权重评估每个视觉 token 的重要性，按预算保留最关键的 r 个 token。
  - **自适应目标特征注入**：从目标模型候选层中随机采样一层，将其输出通过交叉注意力注入草稿模型。
  - **AIFD**：计算每个中间层的注意力熵 AE(ℓ) 和层间变化 ∆AE(ℓ)，选择两者之和最小的层 ℓ* 作为蒸馏目标，提供稳定且信息丰富的监督信号。
  - **损失函数**：包含三项：KL 散度 L_KL（输出分布对齐）、中间特征蒸馏 L_distill（早期层特征对齐）、最终层特征对齐 L_feat，权重 λ_feat=λ_distill=0.2，λ_KL=1.0。
- **算法流程（文字说明）**：
  - Phase 1（热身训练）：完整训练草稿模型超网络，使用 AIFD 选择蒸馏层，通过交叉注意力融合目标特征。
  - Phase 2（多分辨率训练）：每轮随机采样某个子网络配置（注意力头数、视觉 token 预算、目标特征注入层），在原训练基础上继续优化子网络。
  - 训练完成后，遍历所有子网络，选择在目标硬件上预估加速比最高的配置作为最终草稿模型。

## 6. 论文的主要结论与发现

- DREAM-S 在四个 VLM 上平均加速比达 2.32×~3.16×，最高达 3.85×（LLaVA-13B, MMT-Bench）。
- 显著优于现有最强基线：比 EAGLE-3 提升 5%~10%，比 DREAM 提升 2%~5%。
- NAS 搜索的每个维度（注意力头剪枝、视觉 token 压缩、自适应特征注入）均贡献正向加速，其中自适应特征注入最敏感。
- AIFD 自适应选择蒸馏层优于任何固定深度，熵与层变化联合指标可提供更好监督。
- 较大模型受益更多（13B 表现优于 7B）；视觉推理任务（MMT-Bench, ScienceQA）加速效果最明显；字符级精度任务（OCRBench）加速相对温和。
- DREAM-S 在零样本设置下仍保持领先，说明增益源于架构优势而非数据过拟合。
- 固定配置跨模型仍能超越基线，但 NAS 自适应配置在细粒度任务上额外提升显著。

## 7. 优点

- **自动化设计**：首次将 NAS 系统性地用于 VLM 推测解码的草稿模型设计，无需手动调参。
- **硬件自适应**：NAS 搜索可针对不同 GPU 选择最优配置，实验验证了在 A100/H100/RTX8000 上的鲁棒性。
- **蒸馏策略创新**：注意力熵+层间变化联合指标选择蒸馏层，比固定深度挖掘更有效的监督信号。
- **训练高效**：两阶段渐进训练只引入少量额外开销（+1h/epoch），搜索为一次性成本。
- **实验完备**：涵盖了多个模型规模、多种基准、多个温度、多项消融、跨硬件和零样本评估，对比方法全面。

## 8. 不足与局限

- **系统级贡献为主**：论文定位为系统层与应用驱动，未提出新的通用 NAS 算法或理论解码原理。
