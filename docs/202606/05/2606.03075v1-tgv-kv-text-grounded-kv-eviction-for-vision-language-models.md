## 摘要
视觉语言模型（VLM）继承了自回归生成范式，并缓存所有先前token的键和值（KV）以加速推理，导致内存消耗随上下文长度线性增长。由于视觉模态中存在大量冗余，这一问题在VLM中尤为突出。尽管KV缓存驱逐方法能有效减少推理内存，但它们在VLM中往往导致显著的性能下降，因为大多数方法是为语言模型设计的，忽略了文本与视觉之间的固有差距。通过系统分析VLM中的模态差距，我们认为视觉信息的重要性应基于文本指导，并因此提出了一种面向VLM的文本引导KV驱逐方法（TGV-KV）。TGV-KV包含三个子模块：（1）文本-视觉预算分配（TVB）基于互信息交互为每层分配预算。（2）文本加权排序（TWR）评估文本优先级，并基于加权文本-图像注意力对视觉重要性进行排序。（3）文本优先保留（TPR）策略战略性地保留文本KV以避免严重的信息损失。我们在五种不同大小和架构的模型上评估了TGV-KV，结果显示，在LLaVA-NeXT上的VizWiz-VQA任务中，TGV-KV保留了99.2%的全KV准确率，并在极端保留预算为5%的情况下将端到端吞吐量提升了52.6%。代码可在https://github.com/Danielement321/TGV-KV获取。

## Abstract
Vision-Language Models (VLMs) inherit the auto-regressive generation paradigm and cache the keys and values (KV) of all previous tokens to accelerate inference, resulting in memory consumption that scales linearly with context length. This issue is particularly pronounced in VLMs due to substantial redundancy in the visual modality. Although KV cache eviction approaches can effectively reduce inference memory, they often incur significant performance degradation in VLMs, as most are designed for language models and overlook the inherent gap between text and vision. By systematically analyzing the modality gap in VLMs in this work, we argue that the importance of visual information should be grounded in textual guidance and accordingly propose a Text-Grounded KV Eviction method for VLMs (TGV-KV). TGV-KV comprises three submodules: (1) Text-Vision Budgeting (TVB) assigns budget to each layer based on the mutual information interaction. (2) Text-Weighted Ranking (TWR) assesses the priority of text and ranks vision importance based on weighted text-image attention. (3) Text-Prioritised Retention (TPR) policy strategically preserves text KV to avoid acute information loss. We evaluate TGV-KV across five models with different sizes and architectures, showing that TGV-KV preserves 99.2% full-KV accuracy on the VizWiz-VQA task with LLaVA-NeXT and boosts end-to-end throughput by 52.6% with an extreme retention budget of 5%. Code is available at https://github.com/Danielement321/TGV-KV.

# 论文总结：TGV-KV: Text-Grounded KV Eviction for Vision-Language Models

## 1. 核心问题与整体含义（研究动机和背景）
- **问题**：视觉语言模型（VLM）在推理时，由于继承了大语言模型的自回归生成范式，需要缓存所有已生成 token 的键值对（KV cache），导致内存消耗随上下文长度线性增长。视觉模态中大量冗余 token 的存在使得这一问题在 VLM 中尤为严重（例如高分辨率图像或视频可包含数万 token）。
- **现有局限**：已有的 KV 缓存驱逐方法大多针对语言模型设计，忽略了文本与视觉之间的 **模态差距**（modality gap），直接迁移到 VLM 会导致严重的性能下降。具体表现为：(1) 视觉 token 间相似性高，文本 token 多样性高；(2) 文本-视觉注意力区域值偏低，且单模态注意力波动大；(3) 累积注意力在模态交界处发生剧烈偏移，导致驱逐极不均匀。
- **目标**：提出一种 **文本引导（Text-Grounded）** 的 KV 驱逐方法 TGV-KV，通过充分利用文本特征来指导视觉 KV 的驱逐，在显著降低内存的同时保持模型性能。

## 2. 方法论：核心思想、关键技术细节
### 核心思想
- **关键观察**（通过实验揭示）：
    1. 不同层对 KV 保留预算的需求不同，跨模态注意力（text-vision）的强度可作为预算分配的指标。
    2. 结合跨模态注意力（TV）与文本内部注意力（TT）能形成最鲁棒的重要性指标；视觉显著性本质上依赖于文本。
    3. 文本 token 对驱逐高度敏感（少量文本 KV 被驱逐即可导致性能崩溃），而视觉 token 具有高冗余性，可被大量驱逐。
- 基于以上观察，TGV-KV 设计了三个协同模块：**TVB**（预算分配）、**TWR**（重要性排序）、**TPR**（优先保留策略）。

### 关键技术细节
1. **Text-Vision Budgeting (TVB)**  
   - 从每层的注意力矩阵中提取 **文本-视觉注意力** \(A^{(TV)}_l \in \mathbb{R}^{N_t \times N_v}\)，计算其和并沿层归一化，得到每层的预算比例 \(b_l\)。公式：  
     \[ b_l = \frac{\sum_{i,j}[A^{(TV)}_l]_{ij}}{\sum_{l'=1}^{L}\sum_{i,j}[A^{(TV)}_{l'}]_{ij}} \]
   - 动态分配预算，而非固定规则。

2. **Text-Weighted Ranking (TWR)**  
   - 对文本 token 计算重要性权重：从文本-文本注意力 \(A^{(TT)}_l\) 中，对每个文本token j，计算其列和并除以位置（去因果掩码影响），得到平均权重 \(\tilde{w}_{l,j}\)。
   - 对视觉 token 的重要性评分：使用文本权重加权文本-视觉注意力，计算公式：  
     \[ s^{(V)}_{l,j} = \sum_{i=1}^{N_t} \tilde{w}_{l,i} [A^{(TV)}_l]_{ij} \]
   - 对文本 token 的重要性直接使用文本-文本注意力的列和 \(s^{(T)}_{l,j}\)。
   - 关键：通过文本加权，使与重要文本（如指令中的关键词）相关的视觉 token 获得更高保留优先级。

3. **Text-Prioritised Retention (TPR)**  
   - 决定每层保留哪些 KV 对：
     - 若预算 \(b_l > N_t\)（可容纳所有文本 KV），则保留全部文本 KV + 从视觉 KV 中选出评分最高的 \(b_l - N_t\) 个。
     - 若预算 \(b_l \leq N_t\)，则仅从文本 KV 中选出评分最高的 \(b_l\) 个（视觉 KV 全被驱逐）。
   - 该策略确保文本信息被优先保留，仅在极高压缩率下才驱逐部分文本 KV。

### 算法流程（文字说明）
- **Prefill 阶段**：计算每层的注意力，提取 TV 和 TT 部分；通过 TVB 分配层预算；通过 TWR 计算每个文本和视觉 KV 的重要性分数。
- **Decode 阶段**：根据 TPR 策略保留重要性高的 KV 对（优先保留文本），其余驱逐。注意：初始 4 个和最后 1 个 token 始终保留（常见实践）。
