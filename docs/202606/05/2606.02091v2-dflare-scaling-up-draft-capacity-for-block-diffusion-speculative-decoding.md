## 摘要
块扩散推测解码通过同时预测一个块内的所有令牌，供目标模型并行验证，从而加速大语言模型推理。一次性预测整个块需要足够强大的草案模型以及有效利用目标模型的内部知识。然而，最先进的方法DFlash将所有草案层限制为共享仅来自少数目标层的单一融合表示，限制了每层的表达能力，并阻碍了草案容量的进一步扩展。在本文中，我们提出\模型名，通过轻量级的逐层融合机制来扩展DFlash狭窄的条件瓶颈：每个草案层以可忽略的开销关注其自身对广泛目标层集合的可学习组合，同时注入更丰富的目标知识并为每个草案层提供独特的输入。这种增强的逐层表达能力使得草案模型能够扩展到更深的结构，并获得一致的性能提升。我们进一步将训练数据从80万样本扩展到240万样本，以充分利用扩大的容量。在涵盖数学推理、代码生成和对话的六个基准测试中，\模型名在Qwen3-4B上实现了平均5.52倍的加速，在Qwen3-8B上实现了5.46倍，在GPT-OSS-20B上实现了3.91倍，分别比DFlash提高了约11%、8%和5%。我们的代码可在https://github.com/Tencent/AngelSlim获取。

## Abstract
Block diffusion speculative decoding accelerates LLM inference by predicting all tokens within a block simultaneously for the target model to verify in parallel. Predicting an entire block at once requires a sufficiently capable draft model and effective utilization of the target model's internal knowledge. However, the state-of-the-art method DFlash constrains all draft layers to share a single fused representation derived from only a few target layers, limiting per-layer expressiveness and hindering further scaling of draft capacity. In this paper, we present \modelname, which flares out the narrow conditioning bottleneck of DFlash through a lightweight layer-wise fusion mechanism: each draft layer attends to its own learnable combination of a broad set of target layers at negligible overhead, simultaneously injecting richer target knowledge and providing every draft layer with a distinct input. This enhanced per-layer expressiveness enables scaling the draft model to deeper architectures with consistent gains. We further scale training data from 800K to 2.4M samples to fully exploit the enlarged capacity. On six benchmarks spanning mathematical reasoning, code generation, and conversation, \modelname attains average wall-clock speedups of 5.52x on Qwen3-4B, 5.46x on Qwen3-8B, and 3.91x on GPT-OSS-20B, improving over DFlash by roughly 11\%, 8\%, and 5\% respectively. Our code is available at https://github.com/Tencent/AngelSlim.

## 1. 核心问题与整体含义
- **研究动机**：大语言模型（LLM）推理加速中，推测解码（Speculative Decoding）通过小型草案模型预测未来令牌，再由目标模型并行验证，减少推理延迟。块扩散推测解码（Block Diffusion Drafting）通过一次前向传播同时预测一个块内的所有令牌，其草案延迟与块大小基本解耦，因此草案模型深度可以扩展。然而，现有最先进方法DFlash将所有草案层的输入共享为一个来自少数目标层的融合表示，导致每层表达能力受限，深度扩展收益递减，无法有效利用更多目标层知识。
- **整体含义**：本文旨在突破DFlash的瓶颈，通过逐层差异化知识注入，解锁草案深度缩放潜力，并借助扩大训练数据进一步释放增强的模型容量，从而显著提升块扩散推测解码的加速效果。

## 2. 方法论
### 核心思想
- **解除DFlash的窄条件瓶颈**：不让所有草案层共享同一融合表示，而是让每个草案层学习自身独有的目标层组合，同时从更多目标层（如9层 vs DFlash的5层）获取信息，实现每层表达能力的最大化。

### 关键技术细节
1. **自适应逐层融合（Adaptive Layer Fusion）**：
   - 对每个草案层 \(i\)，学习一个T维可学习权重向量 \(\alpha^{(i)} \in \mathbb{R}^T\)，经softmax归一化后对目标层隐藏状态 \(h_t^{(j)}\) 加权求和，再经过RMSNorm得到该层专用的融合特征 \(\mathbf{f}_t^{(i)}\)。
   - 融合权重矩阵仅含 \(D \times T\) 个标量参数（例如D=7, T=9共63个），推理时可预计算，几乎不增加延迟。

2. **异构KV投影（Heterogeneous KV Projections）**：
   - 将目标上下文特征与草案自身隐藏状态分别投影到不同的Key/Value子空间（\(\mathbf{W}_t^K, \mathbf{W}_t^V\) 用于目标特征；\(\mathbf{W}_d^K, \mathbf{W}_d^V\) 用于草案块）。查询仍由草案块投影得到。这使两种信息在注意力中拥有独立表示空间，避免干扰。

3. **渐进式位置加权损失（Progressive Position-Weighted Loss）**：
   - 损失函数对块内每个位置k使用指数衰减权重 \(w_k = \exp(-(k-1)/\gamma)\)，但\(\gamma\)从一个较小值\(\gamma_0\)线性增长到\(\gamma_{\max}\)，实现课程学习：早期聚焦前面易优化位置，后期关注后面难位置。公式：\(\gamma(s) = \gamma_0 + (s/S)(\gamma_{\max} - \gamma_0)\)。

### 流程概述
- 训练阶段：随机采样多个锚点位置，每个锚点及其后B-1个掩码位置构成一个块，使用稀疏注意力（块内双向，块间不关注），仅训练草案Transformer层，共享目标模型词嵌入和LM头并冻结。
- 推理阶段：逐轮进行草案-验证循环。草案模型以目标最新令牌及全上下文融合特征为输入，一次生成B-1个候选令牌；目标模型并行验证，接受最长前缀，并输出下一个起始令牌。
