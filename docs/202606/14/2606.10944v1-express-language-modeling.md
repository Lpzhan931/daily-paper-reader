## 摘要
我们引入了一个新工具Express，用于将非因果注意力近似转换为具有匹配近似保证的因果近似。当与最先进的Thinformer近似结合时，Express改进了已知的最佳因果注意力保证，对于长度为n的序列，仅需O(s)内存和O(s^2 log^2(n))压缩开销，即可实现log^{3/2}(n)/s的近似误差。我们将这些进展与高效的I/O感知Triton实现相结合，展示了比FlashAttention 2显著的速度提升，并使用Express克服了语言建模流程中的四个资源瓶颈：长上下文预填充、KV缓存压缩、长序列内存受限解码和长序列计算受限解码。

## Abstract
We introduce a new tool, Express, for converting a non-causal attention approximation into a causal approximation with matching approximation guarantees. When combined with the state-of-the-art Thinformer approximation, Express improves upon the best known causal attention guarantees, delivering $\log^{3/2}(n)/s$ approximation error with only $O(s)$ memory and $O(s^2 \log^2(n))$ compression overhead for a sequence of length $n$. We pair these developments with an efficient I/O-aware Triton implementation, demonstrate substantial speedups over FlashAttention 2, and use Express to overcome four resource bottlenecks in the language modeling pipeline: long-context prefill, KV cache compression, long-form memory-constrained decoding, and long-form compute-constrained decoding.

- **核心问题**：在语言模型（LM）推理中，因果注意力（causal attention）的计算复杂度随序列长度 \(n\) 呈二次方增长，导致长上下文预填充、KV 缓存、长序列解码等环节出现严重的内存和计算瓶颈。现有的非因果注意力近似方法（如 Thinformer）虽然高效，但无法直接应用于因果掩码场景，而现有的因果近似方法（如 HyperAttention、BalanceKV）在误差、内存或运行时间上仍有较大改进空间。
- **整体含义**：本文提出一种通用元过程 **Express**，能够将任意高质量的非因果注意力近似（通过子抽样/减半算法）高效转换为流式因果近似，并保持理论保证。与最先进的 Thinformer 结合后，得到了 **Thinformer Express**，在理论保证和实际性能上均优于现有方案，从而显著降低了 LM 推理的资源消耗。

### 2. 论文提出的方法论：核心思想、关键技术细节、公式或算法流程
- **核心思想**：利用“减半（halving）”算法构建可增量更新的加权缓存，将离线非因果近似转化为在线因果近似。  
- **关键技术**：
  - **Express 算法（Algorithm 1）**：包含三个阶段：
    1. **精确阶段**：直接缓存前 \(n_{\text{out}}\) 个点。
    2. **稀疏阶段**：后续点按批次（大小为 \(2m n_{\text{out}}\)）先均匀子抽样（Subsample），再通过 **Compress2**（Algorithm 2）逐层减半至 \(n_{\text{out}}\)，并入缓存。
    3. **减半阶段**：当缓存大小达到 \(4n_{\text{out}}\) 时，连续应用两次 HALVE 减至 \(n_{\text{out}}\)，并增大减半因子 \(m\)。
  - **Thinformer Express（Algorithm 4）**：以 KH-Express（核减半算法）为底层，使用注意力核 \(\kappa_{\text{att}}\) 维护加权缓存。每个新 token 先通过 **WtdAttention**（Algorithm 5）仅对缓存和自身做加权注意力，然后更新缓存。
- **理论保证**：
  - **Theorem 3**：在概率至少 1/2 下，误差上界为  
    \[
    \|\hat{o}_n - o_n\|_\infty = O\left(\frac{\exp(2R_n^2/\sqrt{d})\sqrt{\log((d+1)n)\log(n_{\text{out}})\log(n_{\text{out}}\log(n/n_{\text{out}}))}}{\sqrt{2^m n_{\text{out}}}} \|V_n\|_{2,\infty}\right),
    \]
    其中 \(n_{\text{out}}=2h\) 为目标缓存大小，\(R_n\) 为最大键/查询范数。
  - 相比 HyperAttention（误差 \(O(n^{-a/6})\)），本方法实现 \(O(n^{-a})\) 误差衰减；相比 BalanceKV，本方法节省空间和压缩开销至少对数因子。
