## 摘要
多模态大语言模型依赖上下文学习进行快速任务适应，但其可扩展性受到有限上下文窗口以及长多模态序列中不断增长的键值缓存成本的严重限制。现有的记忆压缩方法通常依赖于刚性令牌移除或样本依赖的重要性估计，这引入了偏差，破坏了语义结构（尤其是视觉表示），并产生了无法适应新查询的静态记忆。我们提出了TASM（任务感知结构化记忆），这是一个无需训练的框架，通过任务感知、结构保持和动态可访问的记忆构建来解决这些限制。TASM采用任务向量引导的压缩，用捕获样本间共享相关性的任务级方向替代样本特定信号。为了保持潜在流形，它通过二分图匹配实现语义感知的令牌合并，聚合令牌而不进行破坏性修剪。最后，TASM将记忆结构化为一个包含紧凑核心记忆和潜在库的层次结构，支持查询自适应的动态检索。评估证实，TASM在高压缩下保持高性能，有效平衡了效率与适应性。

## Abstract
Multi-modal large language models (MLLMs) depend on in-context learning (ICL) for rapid task adaptation, but their scalability is severely limited by finite context windows and the growing cost of key-value (KV) caches in long multi-modal sequences. Existing memory compression approaches typically rely on rigid token removal or sample-dependent importance estimation, which introduces bias, disrupts semantic structure, particularly for visual representations, and yields static memories that cannot adapt to new queries. We introduce TASM (Task-Aware Structured Memory), a training-free framework that addresses these limitations through task-aware, structure-preserving, and dynamically accessible memory construction. TASM employs task-vector guided compression to replace sample-specific signals with a task-level direction that captures shared relevance across demonstrations. To preserve the underlying manifold, it applies semantics-aware token merging via bipartite graph matching, aggregating tokens without destructive pruning. Finally, TASM structures memory into a hierarchy comprising a compact Core Memory and a Latent Bank, facilitating query-adaptive dynamic retrieval. Evaluations confirm TASM maintains high performance under heavy compression, effectively balancing efficiency with adaptability.

## 1. 核心问题与整体含义（研究动机和背景）
- **背景**：多模态大语言模型依赖上下文学习（ICL）进行快速任务适应，但面临两大瓶颈：有限上下文窗口和长多模态序列中键值缓存的巨大内存与延迟开销。近期“多示例”ICL显示性能随演示数量对数增长，然而处理数千张图像导致KV缓存膨胀，超出GPU显存。
- **现有方法局限**：现有记忆压缩方法（如EMLoC、SnapKV）通常采用刚性令牌修剪（hard pruning）或样本依赖的重要性估计，导致三个结构性问题：(1) **样本特定偏差**——基于答案注意力的评分过拟合演示样本，忽略对测试查询关键但无关演示的上下文；(2) **拓扑破坏**——硬修剪破坏视觉令牌的二维空间和语义结构，损害空间定位类任务；(3) **静态记忆刚性**——压缩后记忆静态不变，无法应对需要细粒度细节的复杂查询。
- **核心问题**：如何在不牺牲多模态语义结构和任务适应性的前提下，实现高效、可扩展的KV缓存压缩，支持动态查询自适应检索。

## 2. 方法论：核心思想、关键技术细节
- **整体框架**：TASM是一个**无需训练**的框架，将记忆压缩从样本特定信号解耦为任务级方向，通过结构保留的令牌合并和动态检索实现高效ICL。
- **三大关键创新**：
  - **任务向量引导压缩**：利用上下文中的问答对，计算问题表示质心与答案表示质心的差值并归一化，得到任务向量（方向表示“问题→答案”的变换）。然后通过正交投影评估每个令牌的重要性（得分结合任务对齐度和值范数），并使用层自适应门控在浅层（依赖局部注意力）与深层（依赖任务向量）之间平滑过渡。
  - **语义感知令牌合并**：将高重要性令牌设为“接收节点”，低重要性令牌设为“源节点”，构造二分图，使用空间约束（视觉令牌只能与空间邻居合并）和余弦相似度作为边权重，求解最优分配，将源节点加权聚合到接收节点，保留信息密度而不进行破坏性剔除。
  - **信息几何动态检索**：构建两层记忆——GPU驻留的**核心记忆**（紧凑压缩）和CPU驻留的**潜在库**（完整但压缩后的残差）。通过计算当前查询注意力分布与上一步分布的JS散度，若超过阈值则从潜在库检索Top‑K相关令牌，动态扩展有效缓存，实现平均$O(1)$复杂度。
- **公式与算法流程**（文字描述）：
  1. 离线阶段：从少量演示中提取任务向量。
  2. 在线推理：对于每个输入令牌，先计算JS散度决定是否触发检索；然后计算基于任务向量的重要性和局部注意力得分；根据重要性将令牌分入核心或源集；通过图匹配将源令牌合并到核心令牌中；核心记忆保留在GPU，未合并令牌存入CPU潜在库。
  3. 复杂度：预填充阶段$O(T \log T)$，解码阶段$O(T \cdot (N_{\text{core}}+N_{\text{ret}}))$，其中$(N_{\text{core}}+N_{\text{ret}})\ll T$。

## 7. 优点
- **创新性**：首次将任务向量概念用于KV缓存压缩（而非模型权重编辑），并结合图匹配合并与动态检索，系统性地解决三个结构局限。
- **无需训练**：直接在预训练模型上运行，无需微调或额外参数，部署成本极低。
- **结构保持**：保留视觉二维拓扑，避免硬修剪的破坏性；空间约束确保合并仅在相邻区域进行，适用于定位类任务。
- **动态自适应**：JS散度驱动的检索只在“不确定”时触发，平衡效率与精度，优于固定周期检索。
- **实验全面性**：覆盖9个基准、多种模型、消融、迁移和效率分析，结论坚实。

## 8. 不足与局限
- **轻微额外开销**：相比硬修剪基线EMLoC，TASM在32k长度下多占用1.3 GB显存（用于潜在库）和35 ms预填充延迟（由于图匹配）。虽然可接受，但对极端低延迟场景可能不友好。
- **检索机制的触发率仍有优化空间**：JS散度阈值需手动设置，且对于某些高频注意力漂移的场景可能触发过密或过低。
- **未考虑流式/在线场景的增量更新**：实验基于离线静态演示集，动态新增演示时的向量重计算效率未讨论。
- **仅验证了7B～8B规模模型**：更大模型（如70B）上的缩放行为未测试，KV缓存压缩效果可能受模型容量影响。
- **视觉令牌合并的图匹配近似**：文中采用贪心算法，未探索更优的全局最优解，对极端复杂场景（如超高分辨率图像）可能有精度损失。
- **潜在偏见**：任务向量基于少量演示问答对计算，若演示有噪声或分布不均衡，可能引入偏见；但实验显示4对即收敛，表明鲁棒。
