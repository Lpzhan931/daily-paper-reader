## 摘要
主流快-慢双系统视觉-语言-动作模型为了效率，将高频动作专家与低频视觉-语言模型解耦，但面临根本性的频率困境：较大的更新间隔导致过时上下文引发的语义漂移，而较小的间隔则削弱了预期的计算节省。此外，由于动作专家仅以单一固定频率接收VLM最后一层表示，丰富的中间特征被丢弃，限制了信息耦合与操控精度。受人类大脑多时间尺度神经处理的启发，我们提出UniFS，一种统一的快-慢架构，通过三个关键设计解决这些挑战。首先，我们将VLM层分为更新频率逐渐降低的组，使浅层捕获快速变化动态，而深层缓存稳定的语义上下文。其次，潜向量反转机制重新路由多尺度VLM特征与动作专家的交互顺序，将快速变化的表示与细粒度动作解码对齐，慢速变化表示与粗略规划对齐。第三，多层次监督策略跨时间尺度实施从粗到细的学习层级。这些设计共同在单一主干内实现更丰富的跨频率信息传递，同时低频路径额外跨步骤保留时间上下文。在LIBERO上的实验表明，UniFS实现了最先进的性能（平均成功率达98.3%，比VLA-Adapter基线提高2.5%），同时将平均推理延迟从36.5毫秒降至17.8毫秒（加速2.1倍）。在Franka平台上的真实机器人实验进一步验证了其实用性。代码已开源：https://github.com/linsun449/UniFS。

## Abstract
Mainstream Fast-Slow dual system vision-language-action models decouple a high-frequency action expert from a low-frequency vision-language model for efficiency, yet they face a fundamental frequency dilemma: large update gaps cause semantic drift from stale context, while small gaps erode the intended computational savings. Moreover, because the action expert receives only the VLM's final-layer representation at a single fixed frequency, rich intermediate features are discarded, limiting both information coupling and manipulation precision. Inspired by multi-timescale neural processing in the human brain, we introduce UniFS, a unified fast-to-slow architecture that resolves these challenges through three key designs. First, we stratify the VLM layers into groups with progressively decreasing update frequencies, enabling shallow layers to capture fast-changing dynamics while deeper layers cache stable semantic context. Second, a latent vector inversion mechanism re-routes the interaction order between multi-scale VLM features and the action expert, aligning fast-varying representations with fine-grained action decoding and slow-varying ones with coarse planning. Third, a multi-level supervision strategy enforces a coarse-to-fine learning hierarchy across temporal scales. Together, these designs enable richer cross-frequency information transfer within a single backbone, while the low-frequency pathways additionally preserve temporal context across steps. Experiments on LIBERO show that UniFS achieves state-of-the-art performance (98.3\% average success rate, a 2.5\% gain over VLA-Adapter baseline) while reducing average inference latency from 36.5~ms to 17.8~ms (2.1$\times$ speedup). Real-robot experiments on a Franka platform further validate its practical applicability. Code is opensourced at https://github.com/linsun449/UniFS.

- **研究动机**：主流VLA模型（如π0、OpenVLA等）采用**Fast-Slow双系统架构**，将高频动作专家与低频视觉-语言模型解耦，以平衡计算效率与控制频率。然而，该方法面临根本性的**频率困境**：
  - 若更新间隔大，过时上下文导致**语义漂移**，影响动作质量；
  - 若更新间隔小，计算节省被削弱，高频优势丧失。
- **附加问题**：动作专家仅接收VLM**最后一层**的单一固定频率表征，丢弃了丰富的中间层特征，导致**信息耦合不足**和**操控精度受限**。
- **背景**：受人类大脑多时间尺度神经处理（如伽马波与贝塔波共存）启发，论文观察到VLM不同层表征的时间变化幅度存在显著差异（浅层变化快、深层变化慢），据此提出统一快慢架构。

## 2. 论文提出的方法论：核心思想、关键技术细节

### 核心思想
- 将快慢动力学**统一集成到单一VLM骨干**中，直接生成**多时间尺度潜向量**供动作专家使用，而非像传统方法将频率绑定到独立模块上。
- 通过**层分组渐进降频**、**潜向量反转**和**多级监督**实现跨频率信息高效传递，并利用低频通道作为隐式记忆。

### 关键技术细节

#### a) 快-慢架构（Fast-to-Slow Architecture, FSA）
- 将VLM的层分为K组（实验中K=5），每组赋予不同的更新频率：浅层频率高（n1=1），深层频率低（nK=16）。
- 更新规则：第k组仅当时间步t是nk的倍数时才重新计算，否则复用前一步的缓存输出。
- 训练时使用**频率特征替换策略（FFR）**：先对所有时间步进行全并行前向计算，再通过可微索引操作将每个特征对齐到其规定频率，从而在保持GPU并行性的同时实现异步更新。

#### b) 潜向量反转（Latent Vector Inversion, LVI）
- 发现标准VLM中深层隐藏状态变化频率高（因靠近动作输出），与FSA要求的深层缓慢更新冲突。
- **反转交互顺序**：初始动作噪声与深层潜向量交互，细化的动作输出与浅层潜向量交互。这样将高频动态迁移到浅层，使深层保持稳定可缓存。

#### c) 多级监督（Multi-Level Supervision, MLS）
- 在每个频率组的动作专家输出上施加辅助损失，损失函数取所有频率组损失的**算术平均**：
  ```
  L_total = (1/K) Σ LL1( f(h_k), A_gt )
  ```
- 低频组接收相同特征但不同时刻动作标签，学习稳定粗粒度规划；高频组学习细粒度即时修正，形成**从粗到细的学习层级**。

#### d) 训练策略：时间批采样
- 从轨迹中随机选取连续窗口，再从中无放回采样K个时间步并排序，保持因果顺序，适用于异步层执行训练。

#### e) 推理加速
- 低频层仅每fk步计算一次，中间步复用缓存；动作专家经潜向量反转后，早期层由慢变输入驱动可降低更新频率。整体理论加速约2.6×。

## 7. 优点

- **生物启发设计**：受大脑多频率协同处理启发，将频率差异从模块级转移到层内，更自然、更高效。
- **解决频率困境**：统一架构避免了显式双系统的硬耦合与固定同步率问题。
- **丰富的特征耦合**：动作专家接入所有层的多尺度特征，而非仅最后一层，增强了信息流。
- **训练与推理解耦**：FFR策略使训练保持全并行，推理时利用缓存获得显著加速，无训练-推理偏差。
- **隐式记忆**：低频路径自然保留历史上下文，无需外部记忆缓冲区。
- **实验全面**：既在标准模拟基准上取得SOTA，又在真实机器人上验证，实验设计细致（包含消融、频率配置、时序分析）。

## 8. 不足与局限

- **训练阶段无加速**：由于FFR需要全并行前向计算，训练时间并未缩短，且需要较大批量以保证时间多样性。
- **分布偏移风险**：FSA和LVI引入的分布变化可能干扰预训练VLM的语义知识，需要精心微调避免性能退化。
- **骨干限制**：当前基于Qwen2.5-0.5B和VLA-Adapter，未验证在更大骨干（如π0）上的有效性；未来需系统研究适应机制。
- **任务范围有限**：仅在LIBERO模拟和三个简单真实任务上实验，未涉及更复杂的长程操作、动态环境或跨具身泛化。
- **记忆能力的量化不足**：虽然声称隐式记忆，但缺乏与传统记忆机制的直接对比实验。
- **频率配置敏感性**：当最大-最小频率比超过24x时性能开始下降，表明极端配置下仍有退化风险。
