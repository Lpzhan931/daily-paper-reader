## 摘要
现代视觉语言模型（VLM）受益于思维链提示和测试时缩放，但由于大型视觉上下文和长解码链，这些收益往往伴随着高昂的推理成本。我们从两个耦合的维度审视这一成本：视觉上下文缩放（VCS），控制传递给语言模型的视觉证据量；以及视觉推理缩放（VRS），控制推理时的推理搜索量。现有方法通常一次优化一个维度，而跨这些维度的计算联合分配尚未得到充分探索。我们提出了自适应视觉推理缩放（AVIS），这是一种轻量级策略，可根据每个查询自适应调整VCS和VRS。AVIS通过关键多样性视觉（KDV）剪枝实现VCS，这是一种免训练的O(N)基于键的规则，用于在预填充前移除冗余视觉标记；并通过自适应自一致性实现VRS，使用学习到的难度预测器选择推理展开的数量。AVIS部署友好，兼容共享预填充推理，其中所有展开重用单个预填充过程和KV缓存。在多种图像和视频推理基准测试中，AVIS在精度-计算权衡上优于仅VCS和仅VRS的基线，并且在基于强化学习后训练的VLM上仍然有效，同时保持低计算量和低延迟。

## Abstract
Modern Vision-Language Models (VLMs) benefit from chain-of-thought prompting and test-time scaling, but these gains often come with prohibitive inference cost due to large visual contexts and long decoding chains. We view this cost through two coupled axes: Visual Context Scaling (VCS), which controls how much visual evidence is passed to the language model, and Visual Reasoning Scaling (VRS), which controls how much inference-time reasoning search is performed. Existing methods typically optimize one axis at a time, leaving the joint allocation of compute across these axes underexplored. We introduce Adaptive Visual Inference Scaling (AVIS), a lightweight policy that adapts both VCS and VRS per query. AVIS realizes VCS through Key Diversity Visual (KDV) pruning, a training-free $O(N)$ key-based rule for removing redundant visual tokens before prefilling, and realizes VRS through adaptive self-consistency, using a learned difficulty predictor to select the number of reasoning rollouts. AVIS is deployment-friendly and compatible with shared-prefill inference, where all rollouts reuse a single prefilling pass and KV cache. Across diverse image and video reasoning benchmarks, AVIS improves the accuracy--compute trade-off relative to VCS-only and VRS-only baselines, and remains effective on top of RL post-trained VLMs while keeping compute and latency low.

## 1. 核心问题与整体含义（研究动机和背景）

- **问题**：现代视觉语言模型（VLM）在受益于思维链（CoT）提示和测试时缩放的同时，推理成本急剧增加。其主要开销来自两方面：一是高分辨率图像/视频产生的大量视觉上下文（Visual Context Scaling, VCS）；二是推理时的搜索过程（Visual Reasoning Scaling, VRS），如多次采样或长链推理。
- **现有不足**：现有方法通常单独优化VCS（如视觉令牌剪枝）或VRS（如自一致性采样），缺乏对两者联合分配计算资源的探索，导致要么计算浪费，要么性能受损。
- **研究目标**：提出一种自适应策略，为每个输入实例联合分配VCS和VRS的计算预算，在保持或提升精度的同时降低整体推理成本。

## 2. 方法论

### 核心思想
- 将推理计算分配视为一个联合优化问题：配置参数θ=(ρ, K)，其中ρ是剪枝比率（控制VCS），K是推理滚动次数（控制VRS）。目标是在总计算不超过基线的前提下最大化准确率。
- AVIS采用两阶段自适应策略：先通过视觉令牌剪枝降低预填充成本，再通过难度感知的自一致性采样分配不同的滚动次数。

### 关键技术细节

1. **视觉上下文缩放（VCS）——关键多样性视觉（KDV）剪枝**
   - 提出一种免训练的、O(N)复杂度的视觉令牌选择规则。
   - 计算每个视觉令牌的注意力键向量与每头平均键向量的余弦相似度，取负值作为冗余度分数；分数越高（即与平均方向差异越大）表示越不冗余。
   - 通过阈值τ自适应确定保留令牌数量（与cos(τ)比较），实现逐样本的剪枝率ρ。

2. **视觉推理缩放（VRS）——自适应自一致性**
   - 使用自一致性（self-consistency）采样K个推理轨迹，通过多数投票聚合答案。
   - 训练一个轻量级难度预测器，从视觉嵌入中预测每个样本的“可解性”得分ˆp(x)。
   - 根据边际效用（inverted-U曲线），将ˆp(x)映射到离散的K ∈ {1,3,5,7}，只有中等难度的样本获得较多滚动，极难或极易的样本使用较少滚动。
   - 预测器由1D卷积、GroupNorm、SiLU、全局平均池化和浅层MLP组成，仅依赖视觉嵌入，可在预填充前运行，开销极低。

3. **共享预填充推理**
   - 所有K个滚动共享同一个预填充过程和KV缓存，显著降低额外计算成本。

## 6. 主要结论与发现

1. AVIS相比Vanilla基线，在图像基准上平均提升3%准确率，同时降低52% FLOPs；在视频基准上提升2.4%准确率，降低66% FLOPs。
2. 与固定联合配置（ρ=75%, K=5）相比，AVIS以40%更低计算量取得更高准确率。
3. 在等计算量对比中，AVIS超过最接近的固定策略3.7%。
4. 在RL后训练的VLM（VL-Rethinker, Vision-R1, OpenVLThinker）上，AVIS依然保持或提升准确率，同时大幅降低计算量。
5. KDV剪枝保留25%视觉令牌时保留97.99%的Vanilla准确率，优于其他免训练剪枝方法。
6. 难度预测器学会非平凡分配：不同基准上平均K从2.12到3.79不等，且集中分配滚动次数在中等难度样本上。
7. 联合自适应VCS和VRS优于单独自适应任一轴。

## 7. 优点

- **方法创新**：首次将VCS和VRS作为耦合资源进行联合优化，提出轻量、部署友好的两阶段自适应策略。

- **广泛验证**：覆盖多种图像/视频推理任务，包括数学、OCR、感知、时间推理等；与多种剪枝和缩放基线严格对比；在RL后训练模型上也有效。
