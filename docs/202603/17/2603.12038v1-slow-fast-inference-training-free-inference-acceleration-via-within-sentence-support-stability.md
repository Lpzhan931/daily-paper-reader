## 摘要
长上下文自回归解码仍然开销巨大，因为每个解码步骤都必须重复处理不断增长的历史记录。我们在解码过程中观察到一个一致的模式：在句子内部，更广泛地在短的语义连贯跨度内，主要的注意力支持（attention support）通常保持基本稳定。基于这一观察，我们提出了快慢推理（Slow-Fast Inference, SFI），这是一种无需训练的解码框架，它将生成过程解耦为频繁的低成本“快步”和偶尔的密集注意力“慢步”。快步复用紧凑的稀疏内存以实现高效解码。慢步在语义边界附近触发。在慢步中，模型重新审视更广泛的上下文，并使用选择器（Selector）刷新选定的内存，供后续快步使用。在评估的各种上下文长度中，SFI 提供了约 1.6 倍至 14.4 倍的更高解码吞吐量，同时在长上下文和长思维链（CoT）设置下，其质量通常与全 KV 缓存基线持平。由于 SFI 无需训练且可直接应用于现有检查点，它为降低当代自回归推理模型在长上下文、长周期和智能体（agentic）工作负载中的推理成本提供了一条切实可行的路径。

## Abstract
Long-context autoregressive decoding remains expensive because each decoding step must repeatedly process a growing history. We observe a consistent pattern during decoding: within a sentence, and more generally within a short semantically coherent span, the dominant attention support often remains largely stable. Motivated by this observation, we propose Slow-Fast Inference (SFI), a training-free decoding framework that decouples generation into frequent low-cost fast steps and occasional dense-attention slow steps. Fast steps reuse a compact sparse memory for efficient decoding. Slow steps are triggered near semantic boundaries. At slow steps, the model revisits the broader context and uses the Selector to refresh the selected memory for subsequent fast steps. Across the evaluated context lengths, SFI delivers approximately $1.6\times$--$14.4\times$ higher decoding throughput while generally maintaining quality on par with the full-KV baseline across long-context and long-CoT settings. Because SFI is training-free and applies directly to existing checkpoints, it offers a practical path to reducing inference cost for contemporary autoregressive reasoning models in long-context, long-horizon, and agentic workloads.

### 1. 核心问题与研究动机
*   **核心问题**：长上下文自回归解码的计算成本极高。虽然 KV Cache 避免了重复计算，但随着生成长度增加，每一步解码仍需对不断增长的历史记录进行全量注意力计算，导致显存带宽和计算开销成为瓶颈。
*   **研究动机**：作者观察到一种“句内支持稳定性”（Within-Sentence Support Stability）现象：在句子内部或短的语义连贯跨度内，模型关注的注意力支撑点（Attention Support）基本保持稳定，只有在语义边界（如标点符号）附近才会发生显著迁移。
*   **目标**：利用这种时间上的稳定性，设计一种无需训练的解码框架，在保证生成质量的前提下大幅提升推理速度。

### 2. 方法论：Slow-Fast Inference (SFI)
*   **核心思想**：将解码过程解耦为频繁的低成本“快步（Fast Steps）”和偶尔的高成本“慢步（Slow Steps）”。
*   **关键技术细节**：
    *   **快慢步调度**：
        *   **快步**：仅对一个紧凑的稀疏状态（包含 Sink 令牌、最近窗口令牌和由选择器选出的长期记忆令牌）进行注意力计算。
        *   **慢步**：当生成特定触发令牌（如句号、问号、换行符）或达到预设的最大步数（如 64 步）时触发。此时执行全量注意力，并刷新稀疏状态。
    *   **Selector（选择器）**：
        *   **闭式融合**：利用慢步产生的全量注意力 Logits 作为“证据”，结合基于 Key 范数（抑制离群值）和位置（防止过度关注末尾）的“先验”。
        *   **逆 KL 散度目标**：通过最小化逆 KL 散度，得到一个解析形式的闭式解，用于融合证据与先验，生成连续的重要性得分。
        *   **得分精炼**：引入 Soft-NMS（非极大值抑制）减少局部冗余，以及跨头排他性（Cross-head Exclusivity）增加不同注意力头之间选择的多样性。
    *   **系统优化**：
        *   **异步流水线**：利用 CUDA 多流技术，将慢步中的 Selector 计算和缓存重组与下一层的注意力计算重叠，隐藏维护开销。
        *   **内存对齐算子**：设计了两段式 KV 布局，将长期记忆令牌打包成连续内存，以实现高效的带宽利用。
