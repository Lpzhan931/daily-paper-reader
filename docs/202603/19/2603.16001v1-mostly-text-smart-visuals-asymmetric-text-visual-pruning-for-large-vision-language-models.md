## 摘要
网络剪枝是实现轻量化大语言视觉模型（LVLMs）的一种有效技术，它主要将权重和激活值都纳入重要性指标中。然而，现有的工作通常以统一的方式处理来自不同模态的校准数据，忽略了模态特定的行为。这提出了一个关键挑战：如何处理文本和视觉 token 的差异化行为，以实现 LVLMs 的精确剪枝。为此，我们通过解耦相应的权重，系统地研究了视觉和文本 token 对剪枝操作的敏感性，结果表明：(i) 文本路径应通过文本 token 进行校准，因为它表现出比视觉路径更高的敏感性；(ii) 视觉路径表现出高度冗余，甚至允许 50% 的稀疏度。受这些见解的启发，我们为 LVLMs 提出了一种简单而有效的非对称文本-视觉权重剪枝方法，称为 ATV-Pruning，该方法通过从文本和视觉路径中选择信息丰富的 token 来建立精确权重剪枝的重要性指标。具体而言，ATV-Pruning 整合了两项主要创新：首先，通过利用所有文本 token 和视觉 token 的子集自适应地构建校准池；其次，我们设计了一种层自适应选择策略来产生重要的视觉 token。最后，在标准多模态基准测试上的广泛实验验证了我们的 ATV-Pruning 优于现有最先进的方法。

## Abstract
Network pruning is an effective technique for enabling lightweight Large Vision-Language Models (LVLMs), which primarily incorporates both weights and activations into the importance metric. However, existing efforts typically process calibration data from different modalities in a unified manner, overlooking modality-specific behaviors. This raises a critical challenge: how to address the divergent behaviors of textual and visual tokens for accurate pruning of LVLMs. To this end, we systematically investigate the sensitivity of visual and textual tokens to the pruning operation by decoupling their corresponding weights, revealing that: (i) the textual pathway should be calibrated via text tokens, since it exhibits higher sensitivity than the visual pathway; (ii) the visual pathway exhibits high redundancy, permitting even 50% sparsity. Motivated by these insights, we propose a simple yet effective Asymmetric Text-Visual Weight Pruning method for LVLMs, dubbed ATV-Pruning, which establishes the importance metric for accurate weight pruning by selecting the informative tokens from both textual and visual pathways. Specifically, ATV-Pruning integrates two primary innovations: first, a calibration pool is adaptively constructed by drawing on all textual tokens and a subset of visual tokens; second, we devise a layer-adaptive selection strategy to yield important visual tokens. Finally, extensive experiments across standard multimodal benchmarks verify the superiority of our ATV-Pruning over state-of-the-art methods.

#### 1. 核心问题与研究动机
随着大语言视觉模型（LVLMs）规模的不断扩大，其在资源受限设备上的部署面临巨大挑战。**网络剪枝**是减轻模型负担的关键技术，但现有方法（如 Wanda, SparseGPT）通常将文本和视觉 Token 视为同质数据进行统一校准，忽略了**模态异质性**。
*   **研究动机**：作者发现文本和视觉路径在剪枝敏感度上存在显著差异。实验表明，文本路径对校准数据高度敏感，必须使用文本 Token 校准；而视觉路径具有极高的冗余性，即使在 60% 的稀疏度下也能保持性能。
*   **核心挑战**：如何在剪枝过程中准确衡量不同模态 Token 的重要性，以实现更优的跨模态对齐和语义保留。

#### 2. 方法论
论文提出了 **ATV-Pruning（非对称文本-视觉剪枝）**，其核心思想是构建一个“以文本为锚点、精简视觉”的非对称校准池。
*   **核心思想**：在计算权重重要性得分时，校准池包含**全部文本 Token** 和**层自适应筛选的视觉 Token 子集**。
*   **关键技术细节**：
    *   **校准池构建**：定义 $S_{cal} = T \cup V_{sub}$，其中 $T$ 为全量文本，$V_{sub}$ 为筛选后的视觉子集。
    *   **视觉漂移（Visual Drift）**：利用 Token 在 Transformer 块输入与输出之间的余弦距离来衡量其“显著性”。漂移越大，说明该 Token 在该层参与的计算越活跃。
    *   **层自适应预算**：根据每一层平均的视觉漂移程度，动态分配该层保留的视觉 Token 数量（$K$）。视觉活动频繁的层保留更多 Token。
    *   **重要性评分**：基于改进的 Wanda 指标，结合权重幅值与上述非对称校准池产生的激活范数来决定剪枝对象。

*   **硬件环境**：所有实验均在**单张 NVIDIA A100 GPU** 上运行。

#### 6. 主要结论与发现
*   **非对称敏感性**：文本路径是 LVLM 的核心，剪枝时必须优先保护；视觉路径冗余度极高，大幅度剪枝对性能影响微乎其微。
*   **校准池质量胜过数量**：盲目增加视觉 Token 会稀释文本信号，导致性能下降。精选少量高显著性视觉 Token 配合全量文本 Token 是最佳策略。
*   **性能领先**：ATV-Pruning 在 60% 稀疏度下的平均性能保留率显著优于现有方法，尤其在复杂推理任务（如 MMMU, ScienceQA）上优势明显。

#### 7. 优点
*   **简单高效**：无需复杂的迭代优化或权重更新，计算开销极低，易于集成到现有管线中。
*   **见解深刻**：通过解耦实验揭示了 LVLM 内部不同模态路径的冗余特性，为后续研究提供了理论依据。
*   **通用性强**：在不同架构（LLaVA 系列、Qwen 系列）和不同剪枝模式（非结构化、半结构化）下均表现出色。

#### 8. 不足与局限
*   **模态覆盖**：目前仅针对图像-文本模态，尚未扩展到音频、视频、深度信息等更多模态的大型多模态模型（LMMs）。

*   **应用限制**：主要关注推理侧的权重剪枝，未涉及训练过程中的稀疏化。
